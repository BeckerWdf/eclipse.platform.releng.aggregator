Map of Java Development Tooling Plug-ins

Last revised 11:00 June 5, 2001

The Eclipse Java development tooling provides a comprehensive Java development environment.

The Java development tooling itself is divided up into a number of separate plug-ins. The following table shows which API packages are found in which plug-ins (the most commonly referenced API packages are highlighted). This table is useful for determing which plug-ins a given plug-in should include as prerequisites.
