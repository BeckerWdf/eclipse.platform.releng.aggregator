JDT User interface

The org.eclipse.jdt.ui plug-in implements the Java specific user interface classes that manipulate Java elements. The following roadmap outlines the major functions in the org.eclipse.jdt.ui package and its sub packages.

  • A Java perspective assembles all of the Java specific UI elements.  Users working in the Java perspective use the customized Java views and editors and have access to all of the workbench actions associated with developing Java code.
  • The Java package view (IPackagesViewPart) shows the package structure of the resources in the workspace and includes menu items for Java specific actions.  The Java type hierarchy view (ITypeHierarchyViewPart) shows the hierarchy for a Java type in the workspace. 
  • A specialized Java editor supports syntax coloring, annotating code with compilation problems, and code assist. A content outliner further decomposes the source file into its underlying elements.  A Java .class file editor is provided that uses a content outliner to decompose the binary file into its underlying elements.
  • Specialized wizards are implemented for creating different kinds of Java elements.  NewJavaProjectWizardPage is a public wizard class that allows other plug-ins to include the Java page when creating their own projects. This wizard page will configure a new project with the Java nature.
