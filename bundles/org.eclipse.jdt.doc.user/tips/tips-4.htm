Package Name Compression

 

  • You can compress the package names displayes in the Package Explorer and Packages views. Select the Window > Preference > Java > Appearance > Compress package name segments checkbox.
  • Compression pattern can contain a digit (n), which represents n characters in the package name.
  • Compression pattern is not used if the resulting name is longer than the original package name. For example, using a pattern '3~.' on a package named 'org.eclipse.jdt' results in 'org.ecl~.jdt'
