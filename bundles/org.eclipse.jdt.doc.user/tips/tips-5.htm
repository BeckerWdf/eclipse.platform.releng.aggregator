Search

 

To search for methods with a certain return type:

  • Open the search dialog
  • Switch to Java Search tab
  • Type in the search expression: '* X' (without the quotes, and with a space between the wild-card and X, where X is your return type)
  • Select Method and Declarations
  • Use a working set to make the search faster.
  • Press Search

You can do global Find and Replace:

  • First do a File search (choose the File tab in the search dialog). Then right-click in the search results in the Search view to see the Replace and Replace Selected pop-up menu items.
