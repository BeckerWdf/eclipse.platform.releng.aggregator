Search

To search for methods with a certain return type:

  1. Open the search dialog.

  2. Switch to Java Search tab.

  3. Type in the Search string: '* X' (without the quotes, and with a space between the wild-card and X, where X is your return type).

  4. Select Method and Declarations.

  5. Use a working set to make the search faster..

  6. Press Search .

 

You can do global Find and Replace:

  1. Do a File search (choose the File tab in the search dialog).
  2. Right-click on an entry in the Search view and use the Replace or Replace Selected from the pop-up menu.
