Templates

  • Templates try to guess the variable name. For instance, if you have an array called foos, the for template will notice this: void m(int[] foos){ for<<press Ctrl+Space here to enter the for - iterate over array template
    }     you get: void m(int[] foos) { for (int i= 0; i < foos.length; i++) { } }
  • You can define your own templates. Go to Window > Preferences > Java > Templates and press the New... button.
  • In template definitions, $ must be escaped by another $. So you write $$Log:$$ if you want $Log:$ to appear in the editor.
