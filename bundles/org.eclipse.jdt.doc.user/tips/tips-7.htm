Editor

 

  • You can speed up typing by:
    • Turning off all before-compiling problem indication (unselect all checkboxes in Window > Preferences > Java > Editor > Problem Indication)
    • Hiding the overview ruler (undelect the Window > Preferences > Java > Editor > Appearance > Show overwier ruler checkbox)
    • Turning off the editor -> Outline synchronization (unselect the Window > Preferences > Java > Editor > Appearance > Synchonize outline selection on cursor move)
  • You can turn off the tooltips by toggling off the Show Text Hover workbench toolbar button (Note: this button is only enabled when a Java editor is open).
  • You can have your copyright notice generated for newly created Java files. Edit the 'filecomment' template (or edit the 'typecomment' template if you need to customize the comment generated for each new type). Note: make sure that the appropriate checkbox(es) under Window > Preferences > Java > Code Generation is/are selected.
