Editor

 

  • You can speed up typing by:
    • Turning off all before-compiling problem indication (unselect all checkboxes in Window > Preferences > Java > Editor > Problem Indication)
    • Hiding the overview ruler (undelect the Window > Preferences > Java > Editor > Appearance > Show overwier ruler checkbox)
    • Turning off the editor -> Outline synchronization (unselect the Window > Preferences > Java > Editor > Appearance > Synchonize outline selection on cursor move)
