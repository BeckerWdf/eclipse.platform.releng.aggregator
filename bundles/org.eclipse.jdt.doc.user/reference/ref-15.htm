Reserved class path variables

Depending on which JRE you select, certain class path variables are set internally by the workbench. The current values for these variables are displayed on the Class path Preferences page. (Workbench > Preferences > Java > Classpath Variables.)

These three variables are reserved and cannot be overwritten:

  • JRE_LIB: The archive with the runtime JAR file for the currently-used JRE.
  • JRE_SRC: The source archive for the currently-used JRE.
  • JRE_SRCROOT: The root path in the source archive for the currently-used JRE.
