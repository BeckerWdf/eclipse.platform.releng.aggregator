Compiler

The compiler preferences are separated in the following sections:

  • Errors and Warnings
  • Classfile Generation
  • JDK Compliance
  • Other

Classfile Generation

Classfile Generation

Option

Description

Default

Add variable attributes to generated class files

If enabled, variable attributes are added to the classfile.

On

Add line number attributes to generated class files

If enabled, line number information is added to the classfile.

On

Add source file name to generated class file

If enabled, the source file name is added to the classfile.

Preserve unused local variables

If enabled, unused local variables are not stripped from the classfile.
