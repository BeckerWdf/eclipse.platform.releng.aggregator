List Pane Commands

Button

Command or Option

Description

Lock View and Show Members in Hierarchy/
Unlock View

Shows the members implementing the selected method in the list pane in the type hierarchy. Only types implementing the method are shown.

When the view is locked, the list pane no longer tracks the selection in the hierarchy pane above.

Show/Hide All Inherited Members

Shows or hides all methods and fields inherited by base classes. When this option is set to Show, the name of the type that defines the method is appended to the method name.

Show/Hide Fields

Show or hide the fields in the list view below the type hierarchy.

Show/Hide Static Members

Show or hide the static fields and methods in the list view below the type hierarchy.

Show/Hide Non-Public Members

Show or hide the static fields and methods in the list view below the type hierarchy.
