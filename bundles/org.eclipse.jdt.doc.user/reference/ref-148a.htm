Type Selection Dialog

Type selection dialogs allow you to browse the workbench for a type.

  • Filter field: In this field, type the first few characters of the type you want to select. You may use wildcards as needed:
    • "*" for any string and "?" for any character
    • "TZ" for types containing "T" and "Z" as upper-case letters in camel-case notation, e.g. java.util.TimeZone, or
    • "NuPoEx" for types containing "Nu", "Po", and "Ex" as parts in camel-case notation, e.g. java.lang.NullPointerException.
  • Matching types: This list displays matches for the expression you type in the filter field.

The behavior of type selection dialogs can be further customized using the dialog menu:
