Detail Pane

The Detail Pane displays detailed information about a selection in the following views:

  • Expressions View
  • Registers View
  • Variables View

The Default Detail Pane displays information in a source viewer (text with source markup). For example, Java objects display toString() values. When alternate detail panes are available for a selection, a "Show Details As" action appears in the context menu allowing the user to select a type of detail pane to display.

There are several commands in the views used to Show the Detail Pane.
