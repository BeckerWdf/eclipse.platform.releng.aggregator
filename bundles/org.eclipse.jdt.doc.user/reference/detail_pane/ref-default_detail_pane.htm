Default Detail Pane

The Default Detail Pane is the displays textual details in the following views:

  • Expressions View
  • Registers View
  • Variables View

The following are the commands available in the default detail pane.

Default Detail Pane Commands

Command

Name

Description

Availability

  Assign Value Allows you to assign the value of the current text in the detail pane to the currently selected variable. Context menu

Copy

Copies the selected text to the system clipboard.

 Context menu
  Find/Replace... Brings up a dialog allowing the text of the detail pane to be searched. Context menu
  Wrap Text Changes if the text in the detail pane should be wrapped or not. Context menu
  Max Length... Opens a dialog allowing you to specify the maximum number of characters to show in the detail pane. Context menu
