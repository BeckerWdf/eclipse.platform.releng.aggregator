Open type

This command allows you to browse the workbench for a type to open in an editor. You can optionally choose to display the type simultaneously in the Hierarchy view.

  • Choose a type : In this field, type the first few characters of the type you want to open in an editor. You can use wildcards as needed (" * " for a string, " ? " for a character).
  • Matching Types : This list displays matches for the expression you type in the Choose a type field.
  • In Packages : This list displays the packages that contain the selected matching type. Select from which package you want to open this type.
  • Show in Type Hierarchy : Check this box if you want to view the type not only in the editor, but also in the type hierarchy view or perspective, as indicated on the Java preferences page (Window > Preferences > Java).
