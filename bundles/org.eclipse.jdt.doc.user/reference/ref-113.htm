Search

This command allows you to search for references to and declaration of the selection. Following a search, the Search view automatically displays.

References

You can search the entire workbench for references to the selection.

Declarations in Workspace

You can search the entire workbench for declarations of the selection.

Declarations in Hierarchy

You can search the type's hierarchy for declarations of the selection.

Implementers

You can search the entire workbench for implementers of the selected interface.
