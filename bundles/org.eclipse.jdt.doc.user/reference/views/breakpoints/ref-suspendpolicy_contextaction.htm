Suspend Policy Context Action

Select the Suspend contextual menu item to change the suspend policy of a breakpoint between suspending all of the threads in the VM and the thread in which the breakpoint occurred.

There is only one menu item visible at any one given time, showing you which suspend policy you will be changing to should you select it.
