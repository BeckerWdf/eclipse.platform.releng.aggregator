Preferences...

Select the Preferences... command to realize the preference pages for variable display options.

There are three pages in total that are shown, all as entries in the tree on the left of the Preferences Dialog.

These pages include:

  • The Java > Debug > Detail Formatters preference page.
  • The Java > Debug > Logical Structrues preference page.
  • The Java > Debug > Primitive Display Options preference page.
