Show Detail Pane

To show the Detail Pane in the applicable view you must choose one of the orientations described in the Layout view menu.

Layout commands that are available include:

  • Horizontal View Orientation [ ]
  • Vertical View Orientation [ ]

If however, you do not want the detail pane visible you simply select the View Only command [ ] to remove the detail pane from the view.

These commands apply to:

  • Expressions View
  • Registers View
  • Variables View
