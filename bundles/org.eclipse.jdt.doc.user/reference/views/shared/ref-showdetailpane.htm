Show Details Pane

To show the details pane in the applicable view you must choose one of the orientations described in the Layout view menu.

Layout commands that are available include:

  • Horizontal View Orientation [ ]
  • Vertical View Orientation [ ]

If however, you do not want the details pane visible you simply select the View Only command [ ] to remove the details pane from the view.

These commands apply to:

  • Expressions View
  • Registers Views
  • Variables View
