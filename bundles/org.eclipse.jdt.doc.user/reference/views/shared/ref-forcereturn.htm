Force Return

Select the Force Return command to have the method in which code is currently suspended immediately return. Execution must be suspended in a Java non-native method. When returning from a method with an expected return value, the currently selected text is evaluated to create a result. If the evaluated result is not the correct type, an error will be displayed. This feature is only available if your Java Virtual Machine supports force early return.

This command applies to:

  • Java Editor
  • Display View
  • Detail Pane (in the Expressions View and Variables View)
