Terminate and Relaunch

Select the Terminate and Relaunch command [ ] to first terminate the selected debug target and secondly, relaunch it.

Once a launch is terminated it can be automatically removed from the debug view via the preference to remove terminated launches when a new one is created on the Run/Debug > Launching preference page.
