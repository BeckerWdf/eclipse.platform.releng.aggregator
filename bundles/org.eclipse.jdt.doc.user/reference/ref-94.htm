View Menu

This button opens a drop-down menu that offers the following commands and options:

  • Show Referenced Libraries : This option can be toggled to display or hide the libraries referenced by the projects in the view.
  • Show Version Info : This option can be toggled to show or hide the labels of repository versions of the resources in the view. Labels are only shown for resources under VCM control.
  • Filters : Filters the view based on the selected pattern(s):
    • .* hides all files that start with " . "
    • *$*.class hides inner classes. This filter is useful when drilling down into the contents of a JAR file to hide inner classes
