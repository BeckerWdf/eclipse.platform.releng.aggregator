Details Pane

The Details Pane shows the toString() method for selected objects. For primitive variables, it shows the value.
Code completion in the context of selected variable or the current stack frame is available as well.

Evaluation can occur on expressions in the Details Pane using Inspect and Display.

The Details Pane is usable within the following views:

  • Expressions View
  • Registers View
  • Variables View
