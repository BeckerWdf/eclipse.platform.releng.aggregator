New Source Folder Wizard

This wizard helps you to create a new source folder to a Java project.

Note that a new source folder can not be nested in an existing source folders or an output location. Therefore, for projects that use the project as source folder, the wizard will suggest to replace this classpath entry with the new created entry and to change the output location.
