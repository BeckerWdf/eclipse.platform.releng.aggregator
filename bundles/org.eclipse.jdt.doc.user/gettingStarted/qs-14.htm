Evaluating expressions

In this section, you will evaluate expressions in the context of your running Java program.

  1. Debug junit.samples.VectorTest.java to the breakpoint in the setUp() method.  (See the Debugging your Programs section for full details.)
    Perform a Step Over operation once or twice to add some values to fFull.
    Open the Display view in the Debug perspective by choosing Window > Show View > Other.... Then select Debug > Display and press OK. In the Display view, type the following line:

    fFull.size()

  2. Select the text you just typed, and from its context menu, select Display.  (You can also choose Display Result of Evaluating Selected Text from the Display view toolbar.)

  3. The expression is evaluated and the result is displayed in the Display view.

  4. On a new line in the Display view, type the following line:

    fFull.toArray()

  5. Select this line, and select Inspect from the context menu.  (You can also choose Inspect Result of Evaluating Selected Text from the Display view toolbar.)
  6. A pop-up opens with the value of the evaluated expression.
