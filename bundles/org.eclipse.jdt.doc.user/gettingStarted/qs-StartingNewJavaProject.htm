Starting a new Java Project

In this section, we will create a new Java project and explain the different components of a Java project. Once you are done, you should be able to handle more complex layouts.

  1. Select the menu item File > New > Project.... to open the New Project wizard.
  2. On the left pane of the first wizard page, select Java, and on the right pane, select Java Project.
    Then click Next.
  3. On the next page, type "MyProject" in the Project name field and click Next.

  4. From the Source tab, click Add Folder....

  5. On the next dialog, click Create New Folder...

  6. Type "src" in the Folder name field and click OK.
    NOTE: If you want to use an existing folder outside of the workspace, you check Link to folder in the file system and browse to the desired folder.

  7. Click OK again.
  8. You will be prompted to know if you want to remove the project as source folder and update the build output folder to "MyProject/bin". The builder writes the generated .class file to the output folder. "MyProject/bin" is considered as the default output folder for the project. Click Yes.

  9. You will end up with this wizard.

  10. If you want to set an output folder per source folder, you need to check Allow output folders for source folders. Specify the desired output folder by double-clicking on this new item.

  11. You can repeat steps 4 to 7 if you want to create multiple source folders.
    NOTE: You might want to create all source folders at step 6, but be sure you select the project name for each source folder. Before clicking OK, you need to select all source folders you created.

    Screen shot with multiple source folders:

  12. You now have a new Java Project in which you can add source files. If you don't want the builder to clean up the output folders during a full build, go to the Properties of your Java Project.
  13. Go to Java Compiler > Use project settings > Build path... and uncheck Scrub output folders on full build.
