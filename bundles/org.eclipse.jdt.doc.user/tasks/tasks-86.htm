Moving a Compilation Unit

This command moves the selected compilation unit and updates all references to it.

To move a compilation unit, select it in the Packages view, and from its context menu, select Move to Another Package.

Parameters page

Move Compilation Unit Parameters Page

  • In the Choose package to move the compilation unit to field, type or click Browse to select the name of the package where you want the compilation unit to move.
  • Click Next to perform a refactoring without preview, or click Finish to perform a refactoring with preview.
