Renaming a class, field, or interface

You can rename a class, field, or interface by modifying its declaration in the compilation unit in which it is declared. However, if you also want to update all references to it, do one of the following:

  1. In the Packages view, double-click the compilation unit where the class, field, or interface is declared, to open it in the editor.
  2. In the Outline view, select the class, field, or interface.
  3. From the class, field, or interface's pop-up menu, select Refactor > Rename.

or

  1. In the Packages view, select the compilation unit where the class, field, or interface is declared.
  2. From the compilation unit's pop-up menu, select Open Type Hierarchy.
  3. Select the class, field, or interface in the Hierarchy view.
  4. From the class, field, or interface's pop-up menu, select Refactor > Rename.
