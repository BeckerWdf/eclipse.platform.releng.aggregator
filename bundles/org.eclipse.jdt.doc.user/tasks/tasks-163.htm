Parameters page

Parameters Page for the Self Encapsulate Field Refactoring Command

  • In the Getter name field, enter the name for the getter.
  • In the Setter name field, enter the name for the setter.
  • Use the Insert new method after combo-box to indicate the location for the getter and/or setter methods.
  • Select one radio button from the Access modifier group to specify the new methods' visibility.
  • In the class in which the field is declared, read and write accesses can be direct or you can use getter and setter.
    • Select the use getter and setter radio button if you want the wizard to convert all these accesses to use getter and setter.
    • Select the keep field reference radio button if you do not want the wizard to modify the current field accesses in the class in which the field is declared.
  • Press Next to perform refactoring with preview or press Finish to perform refactoring without preview.
