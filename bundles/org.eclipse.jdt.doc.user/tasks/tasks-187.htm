Finding Next or Previous Match


To find the next match:

  • In the text (or Java) editor, press Ctrl+K or select Edit > Find Next from the menu bar
  • The next occurrence of the text selected in the editor will be found.

To find the previous match:

  • In the text (or Java) editor, press Ctrl+Shift+K or select Edit > Find Previous from the menu bar
  • The next occurrence of the text selected in the editor will be found.
