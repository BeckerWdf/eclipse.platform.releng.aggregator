Reordering method parameters

You can reorder method parameters.
  • Select the method in a Java view (e.g. Outline, Package Explorer, Members).
  • Do one of the following to open the Modify Parameters refactoring wizard:
    • From the menu bar, select Refactor > Modify Parameters or
    • From the method's pop-up menu, select Refactor > Modify Parameters
