Opening a type in the Packages view

You can open the Packages view on any type that is included on a project's class path.

  1. From the pop-up menu in the Packages view, select Go To > Type. The Open Type page opens.
    Note: If a type name is used for both an interface and a class, the icon shown in the upper pane in the dialog will be either the interface icon or the class icon for both types.

  2. In the Choose a type field, begin typing an expression to narrow the list of available types, using wildcards as needed. As you type, the list is filtered to display only types that match the current expression.

  3. In the Matching Types list, select a type. When there is more than one definition of a type, then in the In Packages list, select a package and project.

  4. Click OK when you are done. The selected type is displayed in the Packages view.
