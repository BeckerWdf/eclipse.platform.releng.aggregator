Using the Externalize Strings wizard


To open the Externalize Strings wizard, do one of the following:

  • Find strings to externalize (using the Find Strings To Enternalize function), select an entry in the resulting dialog and press the Externalize button or
  • Selec the compilation unit in which you want to externalize strings and selecting Source > Externalize Strings from the menu bar or
  • In the Java editor, select Source > Externalize Strings from the pop-up menu

 Note: Externalizing strings is undoable (with the same restrictions as for refactorings). Use Refactor > Undo from the menu bar to undo externalizing strings.
