Copying and moving Java elements

To copy or move most Java elements:

  1. Select the Java element you want to copy or move.

  2. From the element's pop-up menu, select Copy or Move.

  3. In the resulting Project dialog, clear the Use default location check box and click Browse to browse to the desired location for the element (or the copy of the element).
