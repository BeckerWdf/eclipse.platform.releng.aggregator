Using Incremental Find


To use Incremental Find:

  • In the text (or Java) editor, press Ctrl+J or select Edit > Incremental Find from the menu bar
  • The workbench status line displays Incremental Find: information which is updated as you type. The editor is now a the Incremental Find mode.
  • As you type, the editor finds the first occurrence of the text updates the selection after each character typed

You can leave the Incremental Find mode by pressing Esc.
