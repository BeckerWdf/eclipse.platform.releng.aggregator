Using Incremental Find

To use Incremental Find:

  1. In the text (or Java) editor, press Ctrl+J or select Edit > Incremental Find from the menu bar
  2. The workbench status line displays Incremental Find: information which is updated as you type. The editor is now in the Incremental Find mode.
  3. As you type, the editor finds the first occurrence of the text and updates the selection after each character typed

You can leave the Incremental Find mode by pressing Esc.
