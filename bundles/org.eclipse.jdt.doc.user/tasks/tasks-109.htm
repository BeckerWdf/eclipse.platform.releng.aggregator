Building manually

Incremental build

The Build command performs an incremental build that compiles all resources modified since the last build. To trigger an incremental build, either:

  • press Ctrl+B
  • from the menu bar, select Project > Build

Full build

The Rebuild All command rebuilds all workbench resources that have an associated builder. You can select Project > Rebuild All to rebuild all workbench resources.

Incremental project build

The Build Project command executes an incremental build limited to one project. This option is available from the pop-up menu of any project resource in the Package Explorer view.

Full project build

The Rebuild Project command rebuilds all resources contained in a project. This option is available from the pop-up menu of any project resource in the Package Explorer view.
