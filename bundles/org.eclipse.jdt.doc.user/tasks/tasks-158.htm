Parameters Page for the Reordering method parameters Refactoring Command

  • Select one or more parameters
  • Use the Move Up and Move Down buttons to reorder the parameters (you can see a signature preview below the parameter list)
  • Press Next to see the preview or Finish to perform the refactoring without seeing the preview

This refactoring reorders parameters in the selected method and all methods that override it.

Also, all references will be updated to use the new order of parameters.
