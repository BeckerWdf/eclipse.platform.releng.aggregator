Specifying location of the Javadoc command

The JDT uses the Javadoc command (typically available in JRE distributions) to generate Javadoc documentation from source files.

To set the location of the Javadoc command:

  • Use the Window > Preferences > Java > Javadoc preference page to enter the absolute (e.g. C:\Java\14\jdk1.4\bin\javadoc.exe) path to the Javadoc command
