Showing and hiding members in Java views

Several Java views (e.g. Outline, Package Explorer, Members) offer filtering of members (fields, types and methods). The filters are available as toolbar buttons. There are 3 member filters:

  • Hide Fields: when toggled on, this filter will cause all fields to be removed from the view.
  • Hide Static Members: when toggled on, this filter will cause all static members to be removed from the view.
  • Hide Non-Public Members: when toggled on, this filter will cause all non-public members to be removed from the view.
Additionally, Package Explorer can display or hide all elements inside compilation units.

To show members in Package Explorer:

  • Select the Window > Preferences > Java > Appearance > Show members in Package Explorer checkbox
To hide members in Package Explorer:
  • Clear the Window > Preferences > Java > Appearance > Show members in Package Explorer checkbox
