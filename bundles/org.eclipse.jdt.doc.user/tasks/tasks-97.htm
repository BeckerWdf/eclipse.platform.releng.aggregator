Parameters Page for the Rename Method Refactoring Command

  • In the Enter new name field, type a new name for the method.
  • If you do not want to update references to the renamed method, deselect the Update references to the renamed elementcheckbox.
  • Click Next to perform a quick refactoring, or click Finish to perform a controlled refactoring.
