Overriding a method

You can override a method in the workbench.

  1. Open the type hierarchy for the type in which you want to add (that is, override) the method.
    Note: It does not matter whether the subtypes, supertypes, or type hierarchy is displayed.

  2. In the list pane of the Hierarchy view, ensure that the Show All Inherited Members button is toggled on in order to see all the inherited members.
    Note: The selection might not show all members if a filter is active.

  3. From the pop-up menu of one of the inherited methods, select Create in class name (where class name is the name of the class in which the method will be created).
