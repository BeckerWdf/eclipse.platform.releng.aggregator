Overriding the default system libraries for a JRE definition

You can override the system library and its corresponding source attachment when adding a JRE.

  1. Begin adding a new JRE definition.

  2. In the Add JRE dialog, clear the Use default library check box.

  3. If the system was able to determine default settings for the JAR and source files for the selected JRE installation, then the two corresponding fields will already be filled in.

  4. Change the JRE JAR field to point to the file containing your Java system classes.

  5. Change the JRE source field to point to the file containing the source for the Java system classes.

  6. When you are finished with this dialog, click OK to create the new JRE definition. The new JRE definition will use the customized paths for its JAR and source file locations.
