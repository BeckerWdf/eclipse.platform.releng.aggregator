Building a Java program

A build command compiles workbench resources. A build command can be triggered in different ways:

  • Building automatically: If autobuilding is turned on (Workbench > Preferences), then an incremental build occurs every time you save a modified workbench resource.
  • Building manually: You can perform a manual build using a project's pop-up menu or the Project menu in the menu bar.
