Opening a package

To reveal a package in the tree of the Packages view:

  1. In the Packages view, select Go To > Package from the pop-up menu. The Goto Package page opens.

  2. Type an expression in the Choose a package field to narrow the list of available packages, using wildcards as needed. As you type, the list is filtered to display only packages that match the current expression.

  3. Select a package from the list, then click OK. The selected package is displayed in the Packages view.
