Going To a Package

You can reveal a package in the tree of the Packages view.

1.

In the Packages view, select Go To > Package from the context menu.

2.

The Goto Package dialog appears.

3.

In the Goto Package dialog, begin typing an expression in the Choose a Package field that will help narrow the list of available packages, using wildcards as needed.

As you type, the list is filtered to display only packages that match the current expression.

4.

Select a package from the list, then click OK.

5.

The selected package is displayed in the Packages view.
