Showing full or compressed package names

Package Explorer and Packages views can show full or compressed package names.

To show full package names:

  • Clear the Window > Preference > Java > Appearance > Compress package name segments checkbox.
To show compressed package names:
  • Select the Window > Preference > Java > Appearance > Compress package name segments checkbox. Note: A compression pattern can contain a digit (n), which represents the first n characters of a package name segment.
    The compression pattern is not used if the resulting name is longer than the original package name. For example, using a pattern 3~. on a package named org.eclipse.jdt results in org.ecl~.jdt
