Creating a new Java class

Use the New Java Class wizard to create a new Java class. There are a number of ways to open this wizard:

  1. Select the container where you want the new class to reside.
  2. Click the Create a Java Class button in the workbench toolbar.

or

  1. Select the container where you want the new class to reside.
  2. From the container's pop-up menu, select New > Class.

or

  1. Select the container where you want the new class to reside.
  2. From the drop-down menu on the Open The New Wizard button in the workbench toolbar, select Other. In the left pane, select Java, and in the right pane, select Java Class.

or

    Click the Open The New Wizard button. In the left pane, select Java, and in the right pane, select Java    Class.

or

    Select the container where you want the new package to reside. Then, select from the menu bar File > New > Java Class.
