Opening the New Class Wizard

To open the New Class Wizard, do one of the following from the Packages view:

  • Select the container where you want the new class to reside. Then click the Create a Java Class button in the workbench toolbar.
  • Select the container where you want the new class to reside. Then, from its context menu, select New > Class.
  • Select the container where you want the new class to reside. Then, from the drop-down menu on the Open The New Wizard button in the workbench toolbar, select Other. Then in the left pane, select Java, and in the right pane, select Java Class.
  • Click the Open The New Wizard button; then in the left pane, select Java, and in the right pane, select Java Class.
  • Select the container where you want the new package to reside. Then, select from the menu bar File > New > Java Class.
