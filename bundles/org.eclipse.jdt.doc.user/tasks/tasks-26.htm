Creating a new source folder

In a project that uses source folders, you can create a new folder to contain Java source code.

  1. In the Packages view, select the project where you want the new source folder to reside.

  2. From the project's pop-up menu, select New > Source Folder.
    Note: This menu command is only available from the pop-up menu of projects that use source folders. This property is set using the Properties option on the project's pop-up menu.

  3. In the Project Name field, the name of the selected project appears. If you need to edit this field, you can either type a path or click Browse to choose a project that uses source folders.

  4. In the Folder Name field, type a name for the new source folder.

  5. Click Finish when you are done.
