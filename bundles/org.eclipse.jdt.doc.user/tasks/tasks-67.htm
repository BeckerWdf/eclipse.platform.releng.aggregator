Formatting files or portions of code

  1. Open a Java editor on the Java file whose source code you want to format (either partially or completely).

  2. Select the region of the code you want to format. If nothing is selected, then all of the editor content is formatted.

  3. From the editor's pop-up menu, select Format. The code is formatted according to your indicated preferences (Workbench > Preferences > Java > Code Formatter).
