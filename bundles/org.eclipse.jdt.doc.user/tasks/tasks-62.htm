Using the Local History

The JDT extends the workbench concept of local history in three ways:

  • A file can be replaced with an edition from the local history not just in the Navigator but also in the Packages view.
  • The JDT allows you to replace individual Java elements (types and their members) with editions from the local history.
  • The JDT allows you to restore Java elements deleted from the workbench that have been kept in the local history.
