Attaching source to a class path variable

When attaching source for a class path variable entry, both the path to the Archive and the Root Path must be defined by variables. Follow these steps to make a source attachment for a variable:
  1. Select the project, and from its pop-up menu, select Properties.
    2. In the Properties dialog, select the Java Build Path page.
     
  2. On the Libraries tab, select the class path variable to which you want to attach source.
    3. Click the Attach Source button to bring up the source attachment page.
     
  3. In Archive Variable Path field, use a variable to define the location of the archive. Use the Variable button to select an existing variable and the Browse button to append an optional path extension.

    4.  
  4. In the Root Variable Path field, enter a variable that stores the root path prefix of the JAR file.

    5. Click Variable to select an existing variable or Browse to append an optional path extension.
     
  5. Click OK.
