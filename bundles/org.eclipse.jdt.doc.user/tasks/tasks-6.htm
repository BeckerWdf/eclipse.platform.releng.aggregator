Showing single elements or whole Java files

To display a segmented view of the selected Java file, click the Show Source of Selected Element Only button in the workbench toolbar (so that it is pressed).

To display a whole (non-segmented) view of the selected Java file, click the Show Source of Selected Element Onlybutton in the workbench toolbar (so that it is de-pressed).

Note: this toolbar button is enabled only when a Java editor is open.
