Using the Context Menus

A Java search can be conducted via the context menu of selected resources and elements in the following views:

  • Search, Outline, and Hierarchy views: The selected Java element in these views can be searched for declarations and references.
  • Java Editor: If the selection in the Java editor can be resolved to a Java element, then you can search for declarations and references. The Search submenu contains all available searches given the selection's content.
  • Packages view: Packages, Java compilation units, and classes in a library can be searched for declarations and references. If the compilation unit or class contains more than one type, a dialog prompts you to choose one.

To conduct a search from a context menu, follow these steps:

1.

Select a Java element (for example a Java compilation unit in the Packages view or a method in the Outline view) or some text in a Java editor.

2.

From the selection's context menu, select Search to navigate to the available Java searches.

After you select a search t perform, the search progress is shown in a dialog.

3.

You may stop the search process by clicking Cancel in the progress dialog.
