Generating accessors

The Java editor allows you to generate accessors ("getters and setters") for the fields of a type inside a compilation unit.

  1. Ensure that the Outline view is visible.

  2. Open a Java editor for the file in which you want to generate accessors. The Outline view displays the contents of the file.

  3. Select the field for which you want to generate accessors.

  4. Select Generate Getter and Setter from the field's pop-up menu. If possible, the accessors are generated, and they are visible in the Outline view and the editor.
