Showing and hiding empty parent packages

To show empty parent packages:
  1. Select the Filters command from the Package Explorer's drop-down menu.
  2. Unselect the Hide empty parent packages checkbox.
To hide empty parent packages:
  1. Select the Filters command from the Package Explorer's drop-down menu.
  2. Select the Hide empty parent packages checkbox.
Note:

As an example, the parent packages created for package org.eclipse.ui, would be:

(default package)
org
org.eclipse
