Showing and hiding empty parent packages

To show empty parent packages:

  1. Select the Filters command from the Package Explorer's drop-down menu.
  2. Ensure that the Hide empty parent packages filter is not selected.
To show libraries:
  1. Select the Filters command from the Package Explorer's drop-down menu.
  2. Ensure that the Hide empty parent packages filter is selected.

Note:
