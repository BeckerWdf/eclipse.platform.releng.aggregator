Deleting a JRE Definition

You can delete Java runtime environment definitions that are available for executing Java programs in the workbench.

1.

Select from the menu bar Window > Preferences > Java > Installed JREs.

2.

Select the definition and click Remove.

3.

Check the box for the definition that you want to use as the default JRE for the workbench.

The values of the three reserved classpath variables (JRE_LIB, JRE_SRC, and JRE_SRCROOT) are displayed. These values are set automatically depending on the JRE you select.
