Restoring a Deleted Workbench Element

1.

Ensure that the Outline view is visible.

2.

Open the compilation unit to which you want to add a previously-removed Java element from the local history.

3.

Activate the editor by clicking its tab in the editor area, and the Outline view shows the content of the Java file.

4.

In the Outline view, select the element to whose container type you want to restore the deleted element.

5.

From the type's context menu in the Outline view, select Add from Local History.

6.

In the upper pane of the resulting dialog, all available editions of the selected element in the local history are displayed.

7.

Select a version in the upper pane to view the content of the selected element.

8.

When you have identified the edition that you want to restore, press OK.

9.

The local history edition is loaded into the editor.
