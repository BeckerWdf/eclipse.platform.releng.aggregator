Changing debugger launch options

  1. Select Window > Preferences > Debug.

  2. Setting the Default perspective for Debug option sets the selected perspective as the default perspective to become active automatically each time a launch configuration is launched in debug mode. This default can be overridden in a launch configuration by setting the Perspective to switch to or open when launched in: > Debug mode option found on the Common tab in the launch configuration dialog.

    If the selected perspective is not open at the time, a perspective is created. If a perspective is created, it will adhere to the workbench preference for new perspectives (Window > Preferences > Workbench > Perspectives) and will either open in the same window or in a new window.

  3. Setting the Default perspective for Run option sets the selected perspective as the default perspective to become active automatically each time a launch configuration is launched in run mode. This default can be overridden in a launch configuration by setting the Perspective to switch to or open when launched in: > Run mode option found on the Common tab in the launch configuration dialog.

    If the selected perspective is not open at the time, a perspective is created. If a perspective is created, it will adhere to the workbench preference for new perspectives (Window > Preferences > Workbench > Perspectives) and will either open in the same window or in a new window.
