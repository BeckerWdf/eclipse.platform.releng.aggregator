Adding a classpath variable entry

To add a classpath variable entry to your Java build path, follow these steps:
    Select the project you want to add a variable class path entry to.

    From the project's pop-up menu, select Properties.

    In the Properties page, select the Java Build Path page.

    On the Libraries tab, click Add Variable for adding a variable that refers to a JAR file.
    Note: The JRE_LIB variable entry is automatically included for each new project. This variable refers to the runtime JAR of the workbench's default JRE.
    The Classpath Variable Selection page appears. This page allows you to browse the existing variable definitions.

    In the Variable Name field, you can select a variable from the list of defined class path variables to add to the class path of this project.

    In the Path Extension field, define the path relative to the variable.

    The Resolved Path field updates with the current value of the variable.

    Click OK.
