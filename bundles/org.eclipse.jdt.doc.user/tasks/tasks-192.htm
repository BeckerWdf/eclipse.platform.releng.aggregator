Showing and hiding override indicators

 

Outline and Type Hierarchy views can show special icons (override indicators) to indicate members that override or implement other members from supertypes.

To show the override indicators:

  • Select the Window > Preferences > Java > Appearance > Show override indicators in outline and hierarchy checkbox

To hide the override indicators:

  • Unselect the Window > Preferences > Java > Appearance > Show override indicators in outline and hierarchy checkbox
