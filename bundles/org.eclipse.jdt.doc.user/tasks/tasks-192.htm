Showing and hiding override indicators

Outline and Hierarchy views can show special decoration icons to indicate members that override or implement other members from supertypes.

To show the override indicators:

  • Select the Java Method Override Indicator checkbox in the General > Appearance > Label Decorations preference page.
To hide the override indicators:
  • Clear the Java Method Override Indicator checkbox in the General > Appearance > Label Decorations preference page.
