Finding strings to externalize

To find strings to externalize:
  • In a Java view (e.g. Package Explorer), select a set (which can contain a single element) of packages, source folders or projects.
  • From the menu bar, select Source > Find Strings to Externalize
  • A dialog comes up with a list of all compilation units that have some not-externalized strings
