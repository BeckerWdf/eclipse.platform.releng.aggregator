Opening a type hierarchy in its own perspective

The default behavior for opening a type hierarchy view is to replace the Hierarchy view that is already open in the perspective, if one if already open. If there is not already an open Hierarchy view, one is opened automatically.

Sometimes, it is useful to look at or work with several type hierarchies in different perspectives. This can be achieved by changing a workbench preference.

  1. From the menu bar, select Workbench > Preferences and select the Java category. The general Java Preferences page opens.

  2. Select the Open a new type hierarchy inside the Hierarchy perspective radio button and click OK.

  3. Open a type hierarchy on a Java element to verify it is opened in a new perspective.
