Filtering elements

To filter elements:

  1. On the Packages view toolbar, click the Menu button to see the filtering options.

  2. Select or clear the Show Referenced Libraries option to display or hide the libraries referenced by projects in the view. (This option is toggled on by default.)

  3. Click the Menu button again and select Filters. The Package Filters page opens.

  4. Select patterns for filtering the view:

  • Select the .* check box to hide system files. (This filter is selected by default.)

  • Select the *$*.class check box to hide inner classes. This filter is useful when drilling down into the contents of a JAR file to hide inner classes.
