Filtering elements

To filter elements:

1.      On the Package Explorer view toolbar, click the Menu button and choose Filters.

2.      Select or clear the filters that you want to apply to the view (read Filter Description to learn about the selected filter’s functionality).

3.      Optionally, you can select patterns for filtering the view. To do so:

1.      Select the Name filter patterns checkbox at the top of the dialog

2.      In the text field below, specify the desired patterns (names matching one of the patterns will be hidden).
