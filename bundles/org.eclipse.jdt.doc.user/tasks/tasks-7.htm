Changing new type hierarchy defaults

New type hierarchies can open in a type Hierarchy view or in a Java Hierarchy perspective. You can indicate which is the default method for opening new type hierarchies on the preferences pages.

  1. Select Workbench > Preferences, and select the Java category. The general Java Preferences page opens.

  2. Select the radio button for the Hierarchy Perspective or the Type Hierarchy View as the default location in the Open a new type hierarchy inside area.
