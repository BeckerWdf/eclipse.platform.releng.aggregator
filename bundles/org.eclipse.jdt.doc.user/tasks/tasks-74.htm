Showing a type's compilation unit in the Packages view

You can navigate from a type to its compilation unit in the Packages view.

  1. In the Hierarchy or Outline view, select a type.

  2. From the type's pop-up menu, select Show in Packages View. If the Packages view is not already open, then it opens in the current perspective. The workbench navigates to the type's compilation unit.
