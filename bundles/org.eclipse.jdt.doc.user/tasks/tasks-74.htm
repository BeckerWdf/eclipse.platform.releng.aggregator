Showing a Type's Compilation Unit in the Packages View

You can navigate from a type to its compilation unit in the Packages view.

1.

In the Hierarchy or Outline view, select a type.

2.

From its context menu, select Show in Packages View.

3.

The workbench activates the Packages view and navigates its display to the type's compilation unit.

If the Packages view is not already open, then it opens in the current perspective.
