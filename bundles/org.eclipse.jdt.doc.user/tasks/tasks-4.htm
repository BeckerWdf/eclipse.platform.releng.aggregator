Showing and hiding inner types

To show inner types:

  1. Select the Filters command from the Packages view drop-down menu.

  2. Ensure that the  *$*.class  filter is not selected.

To hide inner types:

  1. Select the Filters command from the Packages view drop-down menu.

  2. Ensure that the  *$*.class  filter is selected.
