Showing & Hiding Inner Types

To hide inner types, select the Filters command from the drop-down menu in the Packages view toolbar. Then make sure that the " *$*.class " filter is selected (i.e., turned on).

To show inner types, select the Filters command from the drop-down menu in the Packages view toolbar. Then make sure that the " *$*.class " filter is deselected (i.e., turned off).
