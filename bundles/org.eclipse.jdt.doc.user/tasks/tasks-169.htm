Renaming a field

You can rename a field by modifying its declaration in the compilation unit in which it is declared. However, if you also want to update all references to it, do one of the following:
  1. In the Package Explorer, double-click the compilation unit where the field is declared, to open it in the editor.
  2. In the Outline view, select the field.
  3. From the field's pop-up menu, select Refactor > Rename.
or
  1. In a Java editor, select a reference to the field (or the field's declaration)
  2. From the editor's pop-up menu, select Refactor > Rename
or
  1. In the Package Explorer, double-click the compilation unit where the field is declared, to open it in the editor.
  2. In the Outline view, select the field.
  3. From the menu bar, select Refactor > Rename.
