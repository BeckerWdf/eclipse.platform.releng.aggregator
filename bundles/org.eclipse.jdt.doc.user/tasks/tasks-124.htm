Choosing a JRE for launching a project

Instead of using the default JRE for running and debugging all Java programs in the workbench, you can specifically assign a JRE for launching an individual project.

  1. In the Package Explorer view, select the Java project.
  2. From the project's pop-up menu, select Properties.
  3. Select the JRE properties page.
  4. Click the Use custom JRE for launching button.
  5. In the list of available JREs, select the JRE you want to use to launch this project and click OK.

Note: Changing the JRE used for running does not affect the way Java source is compiled. You can adjust the build path to compile against custom libraries.
