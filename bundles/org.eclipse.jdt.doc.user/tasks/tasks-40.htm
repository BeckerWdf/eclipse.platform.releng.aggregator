Opening a type hierarchy on the current text selection

To open a type hierarchy on a text selection:

Select a Java element or some text in the editor, and do one of the following:

  • Press F4
  • select Open Type Hierarchy from the editor's pop-up menu.

Note: If the selected Java element (or editor selection) is not a type or a compilation unit, the hierarchy will be open on the type in which the current selection lies.
