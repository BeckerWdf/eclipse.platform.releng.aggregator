Creating a Java project to match an existing layout

Several possibilities exist for projects migrating to Eclipse. This example assumes the source files are stored in a separate location outside of your workspace.
  1. From the main workbench window, click File > New > Project. The New Project wizard opens.
  2. Select Java from the left pane and Java Project from the right pane, then click Next. The New Java Project wizard opens.
  3. In the Project name field, type a name for your new Java project and click Next.
  4. On the Source tab, click the Add Folder button.
  5. Click the Create New Folder button to create a new source folder.
  6. Type a name for your new source folder and click Advanced.
  7. Enable the option Link to folder in the file system.
  8. Browse to the location of your existing source files and click the OK button until you return to the New Project Wizard.
  9. Optionally, expand the new source folder to exclude some sub-folders which may contain test files, for example. Select the Exclusion filters and click Edit.  Add the sub-folder names which should be ignored when walking this source folder.
  10. Click Finish when you are done.
By following these steps for multiple source folders with their own unique settings in several projects, you can create very complicated project layouts from the same source tree.

Example 1 

	AllProducts
		Product1
			JavaSourceFiles
				P1Main.java
		Product2
			JavaSourceFiles
				P2Main.java
Create the first project adding a source folder that is linked to /AllProducts/Product1/JavaSourceFiles. The second project would link its source folder to /AllProducts/Product2/JavaSourceFiles.

Example 2 

	AllJavaSourceFiles
		com
			product1
				P1Main.java
			product2
				P2Main.java
Create the first project with a source folder that is linked to /AllJavaSourceFiles, excluding com/product2. The second project would link its source folder to /AllJavaSourceFiles, excluding com/product1.

Example 3 

	AllJavaSourceFiles
		com
			product1
				P1Main.java
		tests
			com
				product1
					tests
						P1Test.java
Create with 2 source folders that are linked to /AllJavaSourceFiles excluding tests, and the second source folder linked to /AllJavaSourceFiles/tests.

Example 4 

	Common
		JavaSourceFiles
			CommonType.java
	Product1
		JavaSourceFiles
			P1Main.java
	Product2
		JavaSourceFiles
			P2Main.java
To create a project for Product1, define 2 source folders that are linked to /Product1/JavaSourceFiles and a second source folder linked to /Common/JavaSourceFiles.
