Opening the New Interface Wizard

From the Packages view, do one of the following:

  • Select the resource where you want the new interface to reside. Then click the Create a Java Interface button in the workbench toolbar.
  • Select the resource where you want the new interface to reside. Then, from its context menu, select New > Interface.
  • Select the resource where you want the new interface to reside. Then, from the drop-down menu on the Open The New Wizard button in the workbench toolbar, select Other. Then in the left pane, select Java, and in the right pane, select Java Interface.
  • Click the Open The New Wizard button; then in the left pane, select Java, and in the right pane, select Java Interface.
  • Select the resource where you want the new package to reside. Then, select from the menu bar File > New > Java Interface.
