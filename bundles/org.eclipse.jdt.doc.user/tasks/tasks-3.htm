Showing and hiding system files

To show system files:
  1. Select the Filters command from the Package Explorer view drop-down menu.
  2. Clear the hide .* files checkbox.

To hide system files:

  1. Select the Filters command from the Package Explorer view drop-down menu.
  2. Select the hide .* files checkbox. This hides files that have only a file extension but no file name, such as .classpath.
