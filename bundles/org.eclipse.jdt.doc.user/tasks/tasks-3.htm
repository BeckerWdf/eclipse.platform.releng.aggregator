Showing and hiding system files

To show system files:

1. Select the Filters command from the Package Explorer view drop-down menu.

2. Ensure that the "hide .*files" filter is not selected.

To hide system files:

1. Select the Filters command from the Package Explorer view drop-down menu.

2. Ensure that the "hide .*files" filter is selected. This hides files that have only a file extension but no file name, such as .classpath.
