Showing & Hiding System Files

To hide system files, select the Filters command from the drop-down menu in the Packages view toolbar. Then make sure that the " .* " filter is selected (i.e., turned on). This will hide files that have only a file extension but no file name, such as .classpath .

To show system files, select the Filters command from the drop-down menu in the Packages view toolbar. Then make sure that the " .* " filter is deselected (i.e., turned off).
