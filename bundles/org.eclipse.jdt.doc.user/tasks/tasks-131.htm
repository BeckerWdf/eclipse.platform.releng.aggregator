Relaunching a Previously-launched Program

The workbench keeps a histories of launched and debugged programs:

The Run and Debug buttons' drop-down menus keep a list of previous launches. To relaunch, select a previous launch from one of these drop-down menus.

Relaunch History List

You can also relaunch a process in the Processes view via the Relaunch command on the process's context menu.

Relaunch from the Processes View

The most recent launch can always be relaunched by selecting from the menu bar Debug > Relaunch Last or by pressing the F10 key.
