Showing and hiding Java files

To show Java files:
  1. Select the Filters command from the Package Explorer's drop-down menu.
  2. Unselect the Hide Java files checkbox.
To hide Java files:
  1. Select the Filters command from the Package Explorer's drop-down menu.
  2. Select the Hide Java files checkbox.
