Java search

The Java searching support allows you to find precise declarations and references of Java elements (packages, types, methods, fields). Searching is supported by an index that is kept up to date in the background as the resources corresponding to Java elements are changed. The Java search does not need the workspace to have ever been built. It is still possible to conduct searches when auto-build is turned off.

The following searches can be conducted from the pop-up menus of Java elements:

Command

Description

References

Finds all references to the selected Java element

Declarations

Finds all declarations of the selected Java element

Implementors

Finds all implementors of the selected Java interface

Read access

Finds all read accesses to the selected Java field

Write access

Finds all write accesses to the selected Java field

 

Searches can be performed on various scopes:

Workspace – all projects and files in them are included in this search

Hierarchy – only the type hierarchy of the selected element (or its declaring type’s) is included in this search

Working Set – only resources that belong to the chosen working set are is included in this search
