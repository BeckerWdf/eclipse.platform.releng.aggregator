Java Perspectives

The Java development tools contribute the following perspectives to the workbench.

Java

A perspective tuned for working with Java projects. It consists of the following views:

  • Packages
  • Type Hierarchy
  • Outline
  • Search
  • Console
  • Tasks

Type Hierarchy

This perspective allows you to explore the type hierarchy of a type. It can opened on types and compilation units and consists of the Hierarchy view and an editor.

Debug

This perspective is tuned for debugging your Java program. It includes the following views.

  • Processes
  • Debug
  • Breakpoints
  • Inspector
  • Variables
  • Display
  • Outline
  • Console
