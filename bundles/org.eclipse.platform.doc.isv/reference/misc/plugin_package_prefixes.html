Using the class loader enhancements

Last update: June 18, 2004

Eclipse 2.1 included several performance enhancements in the area of classloading. The new runtime/classloader design in Eclipse 3.0 supercedes these enhancements. As such, the 2.1 <package-prefixes> mechanism and related command line options (i.e., -noPackagePrefixes and -classloaderProperties) are no longer used or interpreted.
