Annotations and rulers

We've seen that a document provider is responsible for supplying an appropriate IAnnotationModel for its documents.  Let's look at some of the classes in the org.eclipse.jface.text.source package that provide support for annotations.

A specialized text viewer, called a source viewer, is provided in order to support visual text annotations in a vertical ruler to the left of the source code.

IAnnotationModel - defines a generic model used by a ruler to show annotations.  The model holds annotations, knows their text position, and listens for changes   

MarkerAnnotationModel - defines a model for annotations that represent a marker on a resource in the workspace.  (See Resource markers for more information on markers.)
