Expanding the Ant classpath

Plug-ins can use extension points to contribute extra JARs to the Ant classpath. The plug-in contributing the JARs is also added to the Ant classpath. As a consequence, classes inside the extra JARs have access to all classes available for the plug-in.

How to contribute

The extra JARs cannot be plug-in libraries; they must be separate JARs. The consequences are that plug-in classes do not have access to the classes provided by these extra JARs. The extraClasspathEntries extension point provides an example of how to specify the extra JARs in the plugin.xml file.
