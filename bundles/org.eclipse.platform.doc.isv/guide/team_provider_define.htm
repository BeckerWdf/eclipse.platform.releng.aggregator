Defining a provider

The org.eclipse.core.resources.natures extension point is used to add a project nature definition.  Here is the markup for the CVS client.

<extension 
	point="org.eclipse.core.resources.natures" 
	id="cvsnature" 
	name="%cvsNature">
	<runtime>
		<run class="org.eclipse.team.internal.ccvs.core.CVSTeamProvider"/>
	</runtime>
	<one-of-nature id="org.eclipse.team.repository-provider"/>
</extension>

The project nature must be declared as one of the nature set "org.eclipse.team.repository-provider."  This ensures that only one repository provider can be assigned to any given project.  The run class for the nature must extend RepositoryProvider.
