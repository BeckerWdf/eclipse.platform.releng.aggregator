Running Ant scripts programmatically

The Ant support built into Eclipse allows plug-ins to programmatically run Ant scripts. This is done via the AntRunner class included in the org.eclipse.ant.core plug-in.

Example

The following code snippet shows an example of how to use the AntRunner:


import org.eclipse.ant.core.AntRunner;
import org.eclipse.core.runtime.IProgressMonitor;

public class Running {

public static void main(String[] args) throws Exception {
	IProgressMonitor monitor = ...
	AntRunner runner = new AntRunner();
	runner.setBuildFileLocation("c:/scripts/build.xml");
	runner.setArguments("-Dmessage=Building -verbose");
	runner.run(monitor);
}
}

If a progress monitor is used, it is made available for the running tasks. See Progress Monitors for more details.
