Guiding the user through tasks

Even when the platform UI filters out unneeded functionality, there is still a steep learning curve faced by the new user. The platform UI introduces several mechanisms for helping the user choose what task needs to be done and guiding the user through the steps in that task. These mechanisms are used by the workbench itself to help guide new Eclipse SDK users to their desired tasks.

  • Intro support allows plug-ins to define sites that help introduce the product to the end user the first time the platform is started. These sites are used to help guide the user to their desired tasks.
  • Cheat sheets can help guide the user through the steps involved in a complicated task and automatically perform some of the steps on behalf of the user.
