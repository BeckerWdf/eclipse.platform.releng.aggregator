Text editor utilities

<in progress >

By subclassing TextEditor, your editor can inherit some useful utility methods.

Preference handling

  • set preference store when initializing editor class
  • override handlePreferenceStoreChanged
  • example inherits from text editor - so all changes made in text editor preference page have impact on the sample

Key bindings

  • can set up scopes by overriding TextEditor.initializeKeyBindings
  • can call AbstractTextEditor.setKeybindingScopes
