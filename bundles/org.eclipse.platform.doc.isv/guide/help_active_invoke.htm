Invoking the action from HTML

To include active help links in your documentation, you must first declare the use of the active help javascript in the HEAD tag of your HTML: 

 <script language="JavaScript" scr="../org.eclipse.help/livehelp.js> </script>

In the body of your documentation, you refer to your action implementation in javascript. 

 <a href='javascript:liveAction(
	"com.example.helpexample", 
	"com.example.helpExample.OpenPreferencesAction")'>Click here for Preferences.</a>

The parameters for liveAction must include first the plug-in that contains the action and then the name of the class that implements the action.  Any other parameters needed by your action should be appended.  Extra parameters are passed as strings to your action class.
