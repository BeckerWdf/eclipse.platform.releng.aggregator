Invoking the action from HTML

To include active help links in your documentation, you must first declare the use of the active help javascript in the HEAD tag of your HTML: 

 <script language="JavaScript" src="../org.eclipse.help/livehelp.js> </script>

<NEED TO CLARIFY WHAT THE PATH ABOVE IS.  LITERAL OR INTER-PLUG-IN??>

In the body of your documentation, you refer to your action implementation in javascript. 

 <a href='javascript:liveAction(
	"com.example.helpexample", 
	"com.example.helpExample.activeHelp.ActiveHelpOpenPreferencesAction",
	"")'>Click here for Preferences.</a>

The parameters for liveAction must include first the plug-in that contains the action and then the name of the class that implements the action.  The third parameter will be passed as a String to the live help action using its setInitializationString method.
