Products extension point

[This document is still under construction.]

The preferred mechanism for defining a product based on the Eclipse platform is to use the org.eclipse.core.runtime.products extension point. In this extension point, a plug-in can declare the name of its product, as well as the id of the application extension that should be run when the product is invoked. Here is a snippet of the plugin.xml defined by org.eclipse.ui.examples.rcp.browser in the Browser example. 


<extension
	id="product"
	point="org.eclipse.core.runtime.products">
	<product
		name="%productName"
		application="org.eclipse.ui.examples.rcp.browser.app">
	...
A product extension is defined whose application id is "org.eclipse.ui.examples.rcp.browser.app". This application id is also defined in the plugin.xml in the org.eclipse.core.runtime.applications extension. 
   <extension
         id="app"
         name="%appName"
         point="org.eclipse.core.runtime.applications">
      <application>
         <run
            class="org.eclipse.ui.examples.rcp.browser.BrowserApp">
         </run>
      </application>
   </extension>
This extension is defined with the same id that is referenced in the application attribute of the products extension. (The fully qualified name, with plug-in prefix, is used, but the id is the same.)

In addition to the application, the org.eclipse.core.runtime.products extension describes product customization properties that are used to configure the product's branding information. This information is described as named properties. The Browser example defines several of these properties: 


	<property
 		name="appName"
		value="%swtAppName"/>
	<property
		name="windowImages"
		value="icons/eclipse.gif,icons/eclipse32.gif"/>
	<property
		name="aboutText"
		value="%aboutText"/>
	<property
		name="aboutImage"
		value="icons/eclipse_lg.gif"/>
	</product>
The possible property names that are honored by the platform for product customization are defined in IProductConstants. See the javadoc for a complete description of these properties and their values.
