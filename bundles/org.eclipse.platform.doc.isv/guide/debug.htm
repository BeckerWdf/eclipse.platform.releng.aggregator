Program debug and launching support

The resource plug-ins in the Eclipse platform allow you to manage a set a resources and use incremental project builders to transform them into other resources or artifacts.  When the resources you are working with are source files for a program, a compiler is used to build the source files into an executable program.  Once the program is built, how does a plug-in invoke it? 

The org.eclipse.debug.core plug-in provides the API that allow you to describe the method for launching a program in order to run or debug it.  The org.eclipse.debug.ui plug-in includes support for user configuration of launch parameters and implementation of powerful debuggers.
