Ant tasks provided by Eclipse

The Eclipse Platform provides some useful Ant tasks and properties that interact with the workspace. They can be used with scripts running inside Eclipse.

eclipse.refreshLocal

This task is a wrapper to the IResource.refreshLocal() method. Example:

<eclipse.refreshLocal resource="MyProject/MyFolder" depth="infinite"/>

The parameter resource is a resource path relative to the workspace and the parameter depth can be either of the following: zero, one or infinite.

eclipse.incrementalBuild

This task is a wrapper to IProject.build() and IWorkspace.build() methods. Examples:

<eclipse.incrementalBuild/>
<eclipse.incrementalBuild project="MyProject"/>
