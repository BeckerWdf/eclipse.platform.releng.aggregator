API Tools File Generation Ant Task

Purpose

This task runs to generate all files required by Api Tooling inside a binary bundle. This task is currently run only during the Eclipse builds to create the .api_description file for all projects that have an Api Tooling nature.

Usage

Description

The name of the ant task is: apitooling.apifilesgenerator. To be used, the jar file apitooling-ant.jar has to be on the ant classpath.

<apitooling.apifilesgenerator
	projectname="..."
	projectlocation="..."
	targetfolder="..."
/>

Parameters

Attribute Description Required
projectname ... Yes
projectlocation ... Yes
targetfolder ... Yes

Examples

	<apitooling.apifilesgenerator
		projectname="D:\eclipse\3.4.1\eclipse"
		projectlocation="D:\eclipse-SDK-I20081118-0800-linux-gtk.tar.gz"
		targetfolder="D:\report\report.xml"
	/>
