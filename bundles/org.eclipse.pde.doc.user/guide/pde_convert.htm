Converting existing Java projects into PDE projects

A PDE project is a Java project that "knows" that it hosts a plug-in and is eligible for plug-in-specific operations. It is possible to convert a regular Java project into a PDE project at any point. For example, you may have some Java classes that you want to package into a library and share with others as a plug-in.

Select the Java project and choose the "Convert to PDE Project" action from the popup menu.  Remember that plug-in projects must have names that are the same as plug-in ids. If your project has a name that is not suitable, you need to rename it first.

The conversion process will create an initial plug-in manifest if it is missing, add the "PDE nature" to the project, and configure all of the PDE-specific builders for the project.
