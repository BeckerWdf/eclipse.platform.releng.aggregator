Respository and Update Site Target Provisioner

PDE provides a Target Provisioner that allows you to browse update site and respositories for installable plug-ins. These plug-ins will be downloaded and added to your target platform when Finish is pressed.

Choosing a download location

Before adding to your target platform, you must choose the location where the plug-ins will be downloaded to.

Be careful when selecting a directory, all plug-ins found in the directory will be added to your target platform. Setting Clear contents of folder before downloading will delete everything in the specified folder when the Finish button is pressed

Selecting plug-ins to download

Under Available Plug-ins there is a list of plug-ins that can be downloaded from a list of known sites. Select the plug-ins you want to add to your target platform and press Finish.

Press Properties... to see more information about what you have selected in the tree. Press Manage sites... to edit the respositories and update sites you want to look in. Enter text into the text box to filter the tree.
