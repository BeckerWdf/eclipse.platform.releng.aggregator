Target Provisioners

PDE provides a Target Provisioners wizard to utilize target provisioners. A target provisioner is a mechanism used to add plug-ins to the current target platform. The wizard is available by going to the Target Plug-ins section of the Window > Preferences... > Plug-in Development > Target Platform preference page and clicking the Add... button.

 

Provisioner Selection

The Available Provisioners table contains all the registered target provisioners. The provisioner's description will display in the text box on the right once it is selected.

PDE provides the following target provisioners:

  • File System - Adds bundles from one or more folders on your file system.
  • Update Site - Adds bundles from one or more Eclipse Update sites.
