Target Definition Editor

Developers can simplify configuring their development environment by utilizing a target definition specified in a target file. This file contains vital environment information including target platform location, plug-in configuration, and target environment settings.

The target definition is entirely managed by PDE to simplify plug-in development. Its format is internal and is not intended to be read by tools other than PDE.

PDE provides a form-based multi-page editor to manage the target definition.
