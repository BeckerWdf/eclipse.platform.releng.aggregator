Overview

The Overview page defines the target definition and provides hot links to Content and Environment information.

 

Target Definition

The Target Name is a translatable string that appears in the target selection combo on the Target Platform preference page.

The Target Location specifies the location to which the target platform location should be set. For convenience, the editor provides an option if The target platform is the same as the host (running) platform. This option should be used if you wish to develop plug-ins for your current hot environment. Users can also specify an absolute Location other than the host. To aid in portability of target definitions, the path utilitize Debug String Substituion variables.

 

Additional Locations

Target definitions allow you to include required plug-ins from Additonal Locations other than the main target location. Plug-ins found in these locations will be included in the target platform. These locations need to be absolute, but can consist of Debug String Substitution variables to enhance portability.
