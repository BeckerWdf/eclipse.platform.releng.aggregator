Content

The Content page defines the contents of the target definition. The contents can consist of both features and plug-ins. The plug-ins which are specified and those that compose the specified feature, will be enabled when the target is loaded.

The Include all plug-ins from the specified target check box has been added to conveniently add any and all plug-ins which are found in the target's locations. This can be helpful when target platforms are frequently adding and removing plug-ins.

 

Plug-ins

Along with the ability to add and remove plug-ins by name, you have two other options for adding plug-ins. Add Working Set allows you to add the plug-ins from a working set into your target definition. You may also use the Add Required Plug-ins button to add additional plug-ins which are required by the list of current plug-ins. You have the option to Include optional dependencies when computing required plug-ins.

 

Features

Use the Add, Remove, and Remove All button to modify the list of included features.
