Creating filters

Filters can be used to remove errors or warnings. They are simple to add or remove. Let's see how we can add a new filter.

  • Go to the Problem view.
  • Select an API Tooling problem
  • Use Ctrl + 1 to trigger the quickfix from the problem view

    or open the corresponding resource and do Ctrl + 1 on the corresponding problem.

  • One of the quickfix proposals will be the creation of a problem filter for the corresponding problem.
  • Select it and the problem filter is added for you.
  • The corresponding resource is rebuilt and the problem is removed from the problem view.
