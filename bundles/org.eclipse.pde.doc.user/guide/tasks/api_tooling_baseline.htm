Setting up a baseline

Once your plug-in is converted to use API tooling, you need to set up a baseline in order to get the compatibility analysis performed for your plug-in.

  1. Do one of the following to open the API baseline wizard:

    • Select the warning in the problem view that refers to a missing baseline (An API baseline has not been set for the current workspace) and use Quickfix (Ctrl + 1)
      • and click Finish.
    • Click Window>Preferences>Plug-in Development>API Baselines

  2. Then click Add Baseline.... This will open the API Baseline dialog.

  3. Give a meaningful name for your baseline
  4. Click Browse... and select the root folder of the Eclipse SDK you want to choose as your baseline
  5. Click OK in the folder dialog
  6. Once the baseline is initialized, click Finish
  7. Click OK and you are all set. If your workspace contains some existing projects, you will be prompted for a rebuild.
