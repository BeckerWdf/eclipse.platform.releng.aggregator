Compare Viewers

The following preferences can be changed on the Compare Viewers page.

Option

Description

Default

Synchronize scrolling between panes in compare/merge viewers

The two comparison or merge viewers will "lock scroll" along with one another in order to keep identical and corresponding portions of the code in each pane side-by-side.

Turn this option off if you do not want the compare/merge viewers to lock scroll.

On

Show pseudo conflicts

Displays pseudo conflicts, which occur when two developers make the same change (e.g., both add or remove the exact same line of code or comment).  

Turn this option on if you want pseudo conflicts to appear in compare browsers.

Off

Show ancestor pane initially

Sometimes you want to compare two versions of a resource with the previous version from which they were both derived. This is called their common ancestor, and it appears in its own comparison pane during a three way compare.

Turn this option on if you want the ancestor pane to always appear at the start of a comparison.

Off

Text Font

Click the Change button to change the default font for the compare viewers.

Courier Regular 10 pt.
