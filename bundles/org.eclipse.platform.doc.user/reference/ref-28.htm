Bookmarks View

This view displays the list of bookmarks in the workbench.  Bookmarks place a marker at a specific line in a resource.

Here is what the Bookmarks view looks like:

 

Delete Bookmark

This command deletes the selected bookmark. It does not delete the resource with which the bookmark is associated.

Go To File

The Go To File command opens an editor containing the file associated with the bookmark in the editor area.  If the bookmark is associated with a specific line in the file, the cursor in the editor is placed at the beginning of that line.
