Search

The Search preference page allows the user to set preferences for search. The Reuse editors to show matches options allows the user to keep using the same editor for search results to reduce the number of open editors. Emphasize inexact matches is an option that allows you to highlight them in the Search view. If the Search engine isn't 100% sure about the match then the match is considered inexact. You may also set the highlight color.
