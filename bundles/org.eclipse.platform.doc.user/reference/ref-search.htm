Search

The Search preference page allows the user to set preferences for searches. The Reuse editors to show matches option allows the user to keep using the same editor for search results to reduce the number of open editors. Emphasize potential matches is an option that allows you to highlight them in the Search view. If the Search engine isn't 100% sure about the match then it is considered a potential match. You may also set the foreground color for potential matches. If you only want to see exact matches you can check Ignore potential matches. The Default perspective for the Search view allows you to define which perspective should be brought to front when there are new search results.
