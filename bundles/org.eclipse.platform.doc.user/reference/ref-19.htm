Team

The Team preferences page contains options which affect the version management Team support.

Option

Description

Default

Use Incoming/Outgoing mode when synchronizing

The Synchronize view can open in incoming mode, outgoing mode, or incoming/outgoing mode.  Turning this option on will open the Synchronize view in incoming/outgoing mode, regardless of what you are doing (e.g., updating, committing, or both).

Off
