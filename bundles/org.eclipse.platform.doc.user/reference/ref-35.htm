Log Console View

The Log Console view shows the output from a running external tool. From the preferences dialog box, you can change the default settings for the Log Console view using the External Tools preference page.

Toolbar

The toolbar of the log console view includes the following buttons.

Show/Hide Output Structure Tree
Some external tools support displaying output in a structured tree format.
Show Output of Selected Element Only/Show Complete Output
Displays only the output applicable for the currently selected element in the output structure tree.
Clear Output
Clears the contents of the log console.
