Ant Console View

You can run Ant from the context menu of any XML file in the workbench.  Select the XML file and look for the Run Ant... menu item.  When you execute the script, all the output goes to an Ant console view.

More information about Ant is available at http://jakarta.apache.org/ant/index.html.
