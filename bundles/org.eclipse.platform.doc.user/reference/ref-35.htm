Log Console View

The log console view displays output from external tools and Ant scripts.

 

Show/Hide Output Structure Tree

When running Ant scripts the Ant output can be displayed in a structured format. Targets and tasks are shown in a tree hierarchy.

Show Output of Selected Element Only/Show Complete Output

When the output structure tree is used this button can be used to restrict the displayed output to the currently selected element in the output tree.

Clear Output

Clears the contents of the log console.
