External Tools

The following preferences can be changed on the External Tools page.

 

Option

Description

Default

Save all modified resources before running an external tool

If this option is turned on, then whenever you run an external tool, the Workbench will automatically save all resources that have been modified.

Off

Preferred output level

The selected output level determines what kind of messages are displayed in the console. You can select one of the following output levels:

  • Information
  • Verbose
  • Debug

Information

Colors

You can assign colors to the following output types.

  • Error (red)
  • Warning (orange)
  • Information (blue)
  • Verbose (green)
  • Debug (black)

Default values are shown in italics

Console font

You can select the font to use for console output.
