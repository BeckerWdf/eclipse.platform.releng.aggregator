Responsive UI

Many Eclipse functions run in the user interface thread, when met with a slow operation, employing the Responsive UI, which allows the threading of sequential code, will enable you to continue working elsewhere in Eclipse.

For example, building a project can sometimes take more than a few minutes, during which time you may wish to continue to use other functions in Eclipse.

While the project is being built, select Run in Background from the Building Workspace window and the Responsive UI will allow you to carry on with other tasks in Eclipse.

For additonal information on the status of the action, click Details.

Without the Responsive UI activated in the user interface thread the user would be locked out of performing any other actions.
