Navigate Menu

This menu allows you to locate and navigate through resources and other artifacts displayed in the workbench.

Go Into

This command refocuses the active view so that the current selection is at the root. This allows web browser style navigation within hierarchies of artifacts.

Go To

This submenu contains actions that locate and select objects in the active view.

Back

This command refocuses the active view so that it displays the set of artifacts that were visible before the last use of "Go Into" or "Go Up One Level". This is analagous to a web browser's "Back" button.

Forward

This command refocuses the active view so that it displays the set of artifacts that were visible before the last use of "Go Back". This is analagous to a web browser's "Forward" button.

Up One Level

This command refocuses the active view so it is rooted at the parent of the current root of the view. This is the reverse of the "Go Into" command.

Next

This command navigates to the next item in a list or table in the active view. For example, when the search results view is active, this navigates to the next search result.

Previous

This command navigates to the previous item in a list or table in the active view. For example, when the search results view is active, this navigates to the previous search result.
