Perspective Menu

This menu allows you to display, hide, and otherwise manipulate the various views, perspectives, and actions in the workbench.

Open

This command opens a new perspective in the default location. This preference can be changed in the Window > Preferences > Workbench page. All of the perspectives that are open within the workbench window are shown on the shortcut bar.

The perspectives you will likely want to open are listed first. This list is dependent on the current perspective. From the Other... submenu you can open any perspective.

 

Show View

This command displays the selected view in the current perspective.  Views you are likely to want to open are listed first. This list is dependent on the current perspective. From the Other... submenu you can open any view.  The views are sorted into categories in the Show View dialog.

Show/Hide Editors

This command can be toggled to show or hide the editor area in the active perspective.

Save As...

This command allows you to save the current perspective, creating your own custom perspective.  You can open more perspectives of this type using the Perspective > Open > Other menu item once you have saved a perspective. 

Customize...

Each perspective includes a predefined set of actions that are accessible from the menu bar and workbench toolbar.  Related actions are grouped into action sets.  This command allows you to customize the current perspective by showing or hiding various action sets. The first three (New Wizards, Perspectives, Views) control which actions appear in as top level items in their respective menus. The last category (Other) controls which action sets are visible in the perspective. 

Reset

Changes the layout of the current perspective to its original configuration.

Previous

This command switches the current perspective to be the previous perspective, one up from current as shown on the shortcut bar.

Next

This command switches the current perspective to be the next perspective, one down from current as shown on the shortcut bar.

Close

This command closes the active perspective.

Close All

This command closes all open perspectives in the workbench window.
