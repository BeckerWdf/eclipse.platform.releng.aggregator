Search Menu

The Search menu allows you to search for expressions within artifacts in the workbench or within the help system.

Search

This action opens the search dialog, from which you can perform a variety of different searches.

File

This action opens the search dialog to the "File" tab. From there you can search for files in the workbench or for particular expressions within files.

Help

This action opens the search dialog to the "Help" tab. From there you can search for topics and expressions in the online help system.
