Capabilities

The Capabilities preference page allows you to enable or disable Java development and plug-in development, it also provides Team resource management options.

Note: Some capability selections require others to be enabled, because they are required, deselecting them will only result in them becoming re-enabled. This is the case when deselecting Java Development and Core Team Support.

Here is what the Capabilities preference page looks like:

When attempting to enable an action after its capability has been disabled in the preferences page, the following Confirm Enablement prompt will appear verifying that you do indeed want to enable the required capability. Click Details to display a description of the capability.

Note: Capabilities are also sometimes referred to as Activities.
