CVS version label syntax rules

Version labels must meet the following criteria:

  • Begin with a letter
  • Contain only letters, numbers, dashes, and underscores
  • Contain no special characters

Periods are not allowed in CVS version labels. It is suggested that you use dashes or underscores instead.

You can change the initial version label that appears in the Version Selected Resources page by selecting Workbench > Preferences > Team > Initial version name. Type the alphanumeric label that you want to use as the initial automatically generated version label for projects that are versioned for the first time.
