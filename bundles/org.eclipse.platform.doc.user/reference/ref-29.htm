Properties View

This view displays property names and values of a selected resource.

Here is what the Properties view looks like:

 

The following information is displayed in the columns of the Properties view:

Property Column

This column displays the names of the properties of a selected resource.

Property Value Column

This column displays the values of the properties of a selected resource.

The following commands are available in the Properties view:

Show/Hide Categories

This command toggles whether or not to display the properties by category.  A number of properties can fall under one category.  If you choose this option the properties will appear grouped hierarchically by category.

Filter Properties

This command toggles whether advanced properties are displayed in this view.  Basic properties are always shown.

Restore Default Values

This command restores the selected property value to its default value.
