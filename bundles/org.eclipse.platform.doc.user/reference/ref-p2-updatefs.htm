Updating from the File System

The following preferences can be changed on the Install/Update > Updating from the File System page:

Option

Description

Default

When I add an unrecognized archive file as an update site

This option lets you configure how the Software Updates dialog handles unrecognized archive files when they are added as an update site. The system needs certain information about software before it can be installed. Typically this software information is packaged along with the software, but sometimes software is packaged without this necessary information. The system can attempt to generate this data by analyzing the software archive. This option allows you to elect to always attempt to generate this data, to report the unrecognized archive as an error, or to always prompt you to decide what to do.

Ask me

When I add a plug-in file as an update site

This option allows you to configure how the Software Updates dialog handles plug-in archives when added as a site. You can elect to automatically install the plug-in, report an error, or to always prompt you to decide what to do.
