Logging Options

This option indicates the level of logging to report.  Help components can report information, warnings, and errors (related to the installation of the component files) to the user.  

Choose from the following logging options:

  • Errors only
  • Warnings and errors
  • Everything, including information-only messages

Internal Help Server Configuration

 In most cases, you will not have to change the default option of Automatic.  

The workbench automatically detects your IP address and uses the best available port.  The HTTP server indicated here stores local help pages; these values are used to start the internal server.  

Options:

  • Automatic
  • Manual - if you specify this option you need to indicate the following additional options:

    Local Help Server IP Address:  must be a valid address on your machine; you may specify multiple addresses.

    Local Help Server Port:  must to be an available port.
