Text Search

In the Search Dialog, the Text Search tab allows you to search for text in the workbench.  You can bring up the Search Dialog by clicking on the Search toolbar button.

Search Expression

In this field, type the expression for which you wish to search.

From the drop-down menu, you can choose to repeat or modify a recent search.

Wildcards

The available wildcards for search expressions are displayed in the search dialog. For example:

  • "*" matches any set of characters, including the empty string
  • "?" matches for any character
  • "\" is the escape for literals; if you want to search for an asterisk, question mark, or backslash character, type a backslash before it to indicate that you are not using these characters as wildcards (e.g., "\*", "\?", or "\\")

Extensions

In this field, select all the file types to search through for the specified expression.

Case Sensitive

Turn this option on if you want the text search to be case-sensitive.
