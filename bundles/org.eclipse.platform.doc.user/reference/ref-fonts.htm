Colors and Fonts

The fonts used for various types of text can be set using the Colors and Fonts preference page.

A Fonts list allows the selection of the kind of text where the font will appear and shows the appearance of the current font setting.

The font setting can be changed either by selecting the font area from the list and clicking Use System Font to choose the Operating System font setting or by clicking Change to open up a font selection dialog.

Here is what the Colors and Fonts preference page looks like: 

The Colours and Fonts text field can be used to filter the contents. Simply type in an entry and any matching results will remain in the tree view.

Descriptions and previews are provided when the Workbench colors and font settings are selected.
