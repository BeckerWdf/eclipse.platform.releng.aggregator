Fonts

The fonts used for various types of text can be set using the Fonts preference page.

A Fonts list allows the selection of the kind of text where the font will appear while a Preview area shows the appearance of the current font setting.

The font setting can be changed either by pressing the Use System Font button to choose the Operating System font setting or pressing the Changn button to open up a font selection dialog.
