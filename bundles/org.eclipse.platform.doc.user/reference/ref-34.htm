Help View

This view displays help on using the Workbench. If you are reading this document, you've probably found this view already.

Here is what the Help view looks like:

 

Books Dropdown List

This list contains all the available books for which there is help. Selecting a books fills the topics list below it.

Contents Tab 

This tab displays an outline of the available Help topics for the selected book.

Related Tab

If you invoke the context sensitive (F1) help, all the related links for that context are displayed in a list on this tab.

Hide Navigation

This is a button that will toggle showing the list of books and their topic list, allowing more room for the help contents page.

Synchronize Navigation

When this button is toggled, selecting items in a help page will cause the corresponding item in the topic tree to be selected.

Back

You can use this option if you have navigated via links to the currently displayed page.  Click on this button to display the previously displayed page.

Forward

You can use this option if you have navigated via links to the currently displayed page and then used the Back button to display a previous page.  Click on this button to display a  previously displayed page.

Print Page

This command prints the currently displayed page.
