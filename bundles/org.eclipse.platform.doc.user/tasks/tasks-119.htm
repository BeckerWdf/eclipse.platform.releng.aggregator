Viewing the history of a stream resource

Prerequisite: You must have a repository location and be authenticated to the server, before you can view the history of a stream resource in the repository.

  1. Switch to the Team perspective or add the Repositories view to the current perspective.

  2. In the Repositories view, expand either the Project Versions or the Streams category, and select the repository location that contains the resource whose history you want to view.

  3. Right-click and select Show in Resource History from the pop-up menu.

The Resource History view shows you the following information about each version of the stream resource:

  • Version - consists of a repository-generated version number followed by a list of additional version information. In CVS these are the tags associated with that version.
  • Date Created.
  • Author.
  • Version Comments (if any).

Note: The CVS server may not support history.  In this case you will not see the history information specified above.
