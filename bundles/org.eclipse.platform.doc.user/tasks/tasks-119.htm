Viewing Stream Resource History

You can view the history of a stream resource in a repository in the Repositories view.

1.

Pre-requisite: Must have a repository location and be authenticated to the server.

In the Repositories view, expand the repository location that contains the resource for which you want to view the history.  

2.

Select the resource for which you want to view the history. 

You can do this in either:

  • a project version in the Project Versions 

  • a project version in a stream in the Streams 

3.

Bring up the context menu.  Select the "Show in Resource History" menu item.

4.

In the Resource History view, you will see the information about each version of the stream resource:

  • Version

The version consists of a repository generated version number followed by a list of additional version information. In CVS these are the tags associated with that version.

  • Date Created
  • Author
  • Version Comments (if any)

Note: The CVS server may not support history.  In this case you will not see the history information specified above.
