Creating a file

To create a file:

  1. In the Navigator view, right-click the project or folder where you want to create the new file.

  2. From the pop-up menu, select New > File.

  3. Specify the name of the file, including file extension (for example, newfile.txt).

  4. Click Finish.

The editor associated with the type of file that you specified opens.
