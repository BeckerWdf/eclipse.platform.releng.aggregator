Files

You can create new files as follows:

1.

From the workbench window select a project or folder.

2.

 From the folder's context menu choose New > File. 

3.

In the File name field, type a name for the new file, including the file extension (e.g., "test.txt").

4.

Click Finish when you are done.

5.

An editor will automatically open on the new file.
