Projects

To create a project proceed as follows:

1.

From the workbench window choose File > New Project...

2.

The New Project Wizard appears.
3. In the left pane select a category and in the right pane select the type of project to create. Click Next.

4.

In the Project name field, type a name for your new project.

5.

Either check or clear the box indicating that you want to use the default location for your new project.

If you clear this check box, then type or browse to select a location for the project.

6.

If you want to set project references, click Next and select the set of projects that this project needs to reference.  Note that Next is only enabled if another project exists in the workbench.

 

7.

Click Finish when you are done.
