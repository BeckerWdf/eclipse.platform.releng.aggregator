Deleting a Bookmark

From the Marker Bar

1.

In the marker bar, access the context menu from the bookmark that you want to delete.

2.

From the bookmark's context menu, select Remove Bookmark.

3.

The bookmark disappears, and the bookmark is also removed from the Bookmarks view.

From the Bookmarks View

1.

In the Bookmarks view list, select the bookmark(s) that you want to delete.

2.

Do one of the following:

  • Press the Delete key on the keyboard;
  • From the bookmarks' context menu, select Delete; or
  • Click the Delete  button on the view's toolbar.

3.

Notice that the bookmark(s) no longer appear in the marker bar and that the bookmarks are removed from the Bookmarks view.
