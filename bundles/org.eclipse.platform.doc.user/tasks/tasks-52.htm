Editing File Externally

To edit a workbench file from outside the workbench, follow these steps.

1.

Navigate in the file system to the workbench's installation directory. Go into the workspace directory and open the file in an external editor.

2.

Edit the file as needed.

3.

When you are done, save the file in the external editor and close the editor as usual.
4. In the Navigator's context menu choose Refresh from Local. This instructs the workbench to check the local file system for any edits that may have been made without its knowledge.

It is important to use Refresh From Local because if the Workbench detects changes to the local file system it will perform any necessary "builds/updates" to process the changes.
