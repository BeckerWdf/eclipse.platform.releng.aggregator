Comparing with Repository Versions

To compare workbench resources with versions in the repository:

 

1.

Select a resource in the Navigator view.

2.

Bring up the context menu.  Select one of the following menu items:

  • Compare With > Stream Contents: compare workbench resource with the version currently released in the stream.
  • Compare With > Version: compare workbench resource with a version in the repository.
  • Compare With > Base Version: compare workbench resource with the version that was taken from the stream during the last synchronization.

3.

In the compare dialog, browse to select the version with which you want to compare the workbench resource.  Notice you can see the differences between the version and the workbench resource in the text compare area.
4.  Click the Done button to close the compare dialog.
