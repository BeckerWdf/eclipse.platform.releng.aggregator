Comparing resources with repository versions

To compare workbench resources with versions in the repository:

  1. Select a resource in the Navigator view.

  2. From the resource's pop-up menu, select Compare with.

  3. Select the appropriate menu item for the comparison you want to perform, Stream Contents, Version, or Base Version. The Compare editor opens.

  4. Browse to select the version you want to compare the workbench resource to. You can view the differences between the version and the workbench resource in the text compare area.

  5. Click Done to close the compare editor.
