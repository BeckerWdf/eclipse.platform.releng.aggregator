Save Resources Before Build

To automatically save all modified resources in the workbench before a manual build is done:

1.

From the workbench, select the Window > Preferences menu item.

2.

In the preferences dialog, select the Workbench page in the left pane. 

3.

Select the checkbox that says "Save all modified resources automatically prior to manual build".

4

Click the "OK" button of the Preferences dialog.
