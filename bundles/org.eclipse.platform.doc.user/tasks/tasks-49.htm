Viewing resource properties

To display the various properties of a workbench resource:

  1. Right-click the resource in the Navigator view.

  2. Select Properties from the pop-up menu.

The kinds of properties that are displayed depend on the specific resource you selected and the components that are installed into the workbench.

You can also see basic properties for a resource by clicking Perspective > Show View > Other > Basic > Properties, when the resource is selected in the Navigator view.
