Expanding the Ant classpath

When using an optional or custom task it is usually necessary to add extra libraries to the classpath. In order to extend the Ant classpath in the workbench, go to the menu Window > Preferences > External Tools > Ant and edit the classpath according to your needs. The Restore Defaults button restores the Ant classpath to its original state.
