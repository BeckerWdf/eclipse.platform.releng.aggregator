Installing components

To install a new component in the workbench:

  1. Click Help > Software Updates. The Software Updates window opens.
  2. Click Add.  The New Components Installation page opens.  
  3. In the Location field, enter a valid location from which workbench components can be obtained.  (For example: http://siteaddr.domain/updates.) Optionally, add a description for the location in the Description field.
  4. Click Add to add the location to the list, then click Next.
  5. From the list of components, select one or more components to install and click Next. A page listing the components displays.
  6. Click Finish on the wizard, then click Finish when the installation completion page is displayed.
  7. Click OK to close the Software Updates page.
  8. Exit and restart the workbench to complete the install.
