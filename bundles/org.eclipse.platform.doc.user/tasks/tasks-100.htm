Versioning Your Project

To make a version of a project:

1.

Pre-requisite: The project you want to version must first be associated with a Stream.  

In the Navigator view, select the project that you want to version to the repository.  

Note: It is often preferable to synchronize resources with the stream before versioning.

2.

Bring up the context menu.  Select the "Team > Version from Workspace..." menu item.

3.

In the Version Selected Resources dialog, choose one of the following options to label the version of the selected project (or projects):

  • Automatic: each project's version label is automatically generated 
  • One Name: the version label for each project is one name that is specified by the user
  • Name Each: for each project, the user is prompted to enter a version label

4.

Click the " OK" button of the dialog.
