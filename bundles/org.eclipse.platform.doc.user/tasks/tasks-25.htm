Opening Perspectives in Non-default Locations

If the default placement of new perspectives is not where you want to open this new perspective, then you can use the keyboard hot key that is currently associated with the desired command. For example (on Windows) by default:

  • To open in the same workbench window: default (no hot key required)
  • To open in a new workbench window: Hold down the Shift key
  • To replace the current perspective: Hold down the Ctrl key

To confirm or change the current hot key settings, look on the workbench preferences page (Window > Preferences > Workbench).

To activate a non-default perspective placement, simply hold down the desired hot key while selecting the kind of new perspective you want to open from either the fast view button or the Perspective > Open option.
