Copying Using the Menus

You can copy resources from one workbench location to another (for example, from one project to another project) using the Copy command on the context menu in the Navigator view.

1.

In the Navigator view, select the resources that you want to copy.

2.

From the context menu, select Copy.

3.

In the Folder Selection dialog, select the project or folder where you want to copy the resources to.

4.

Choose Ok
