Copying resources

You can copy resources from one Workbench location to another (for example, from one project to another project).

  1. In the Navigator view, select the resources that you want to copy.
  2. From the pop-up menu, select Copy.
  3. In the Navigator view, select the project or folder where you want to copy the resources to.
  4. From the pop-up menu, select Paste.

Tip: You can also copy resources by holding down the Ctrl key and dragging them from the original location to the new location in the Navigator view.
