Copying resources

You can copy resources from one workbench location to another (for example, from one project to another project).

  1. In the Navigator view, select the resources that you want to copy.

  2. From the pop-up menu, select Copy.

  3. In the Folder Selection window, select the project or folder where you want to copy the resources and click OK.

Tip: You can also copy resources by holding down the Ctrl key and dragging them from the original location to the new location in the Navigator view. A horizontal line will appear to show you where the resource will be moved to. You must drag the resource to the exact location where you want it moved.
