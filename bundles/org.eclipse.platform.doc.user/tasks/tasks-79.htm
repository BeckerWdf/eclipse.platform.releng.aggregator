Filtering the Task List

You can filter the tasks that are displayed in the Tasks view.

Filtering by Type

You can choose to show or hide various kinds of tasks (e.g., problems, tasks, etc.) by clicking the Filter button and selecting various types of tasks in the Types of entries list.

Filtering by Hierarchy Level

You can choose to show or hide the tasks related to various levels in the hierarchy by clicking the Filter button and clicking one of the radio buttons.

  • On any resource: Tasks associated with all resources are displayed.
  • On selected resource only: Only tasks associated with the currently-selected resource are displayed.
  • On selected resource and its children: Only tasks associated with the currently-selected resource and its children are displayed.

Filtering by Description Text

You can choose to show or hide the tasks containing a specified string in their descriptions by clicking the Filter button and typing a string in the Where description field and selecting to display tasks whose descriptions either contain or do not contain the string.

  • If you choose "contains", then only tasks whose description contain the string are displayed.
  • If you choose "does not contain", then only tasks whose description does not contain the string are displayed.

Filtering by Problem Severity

You may choose more than one of these options at once.

Note: If Problems are not selected in the Show entries of type list in the top part of the dialog, then this option is not available.

  • Error
  • Warning
  • Info

Filtering by Task Priority

You can indicate what priority of tasks you want to display by clicking the Filter button. You may choose more than one of these options at once.

Note: If Task is not selected in the Show entries of type list in the top part of the dialog, then this option is not available.

  • High
  • Normal
  • Low

Filtering by Task Status

You can indicate what status of tasks you want to display by clicking the Filter button. You may choose more than one of these options at once.

Note: If Task is not selected in the Show entries of type list in the top part of the dialog, then this option is not available.

  • Completed
  • Not Completed
