Specifying the Default Perspective

The Resource perspective is the predefined default perspective. However, you can define a different perspective as the default perspective. The default perspective always shows up on the Perspective > Open menu.

1.

Select from the menubar Window > Preferences.

2.

Expand the Workbench item on the left and select the Perspectives preferences page.

3.

Select from the list the perspective that you want to define as the default.

4.

Click Set Default. The (default) indicator moves to the perspective you just selected.

5.

Click OK.
