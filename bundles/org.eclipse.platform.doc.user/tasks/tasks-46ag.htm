Finding a resource quickly

To navigate to a particular resource:

  1. Right-click in the Navigator view.

  2. From the pop-up menu, select Go To > Resource.

  3. In the window that opens, start typing the name of the resource in the Pattern field. As you type the file name, the system offers a list of possible matches, based on what you have entered so far.

  4. Select the resource that you want from the Matching Resources list, and click OK.

The Navigator view displays the selected resource.

You can also do a contextual search for character strings contained within files in the workbench. See the links to related tasks below.
