Discovering branch and version tags

Prerequisite: Before you can discover the branch and version tags for a repository, you must have a valid repository location in the CVS Repositories view.

To discover existing branch and version tags:

  1. Switch to the CVS Repository Exploring perspective or add the CVS Repositories view to the current perspective.

  2. Select the repository location or folder whose tags are to be discovered.

  3. From the pop-up menu for the CVS Repositories view, select Configure Branches and Versions. The Tag Configuration dialog will open.

  4. In the top pane on the left (titled Browse files for tags), expand one or more projects to find a file that is know to have tags that should be added to the CVS Repositories view.

  5. Select a file in the left top pane. If the file selected has tags defined on it, they will be displayed in the right top pane (titled New tags found in selected files).
  6. Check or uncheck the tags in the right pane as desired.
  7. Click Add Checked Tags to add the checked tags to the list of remembered tags.
  8. Click OK to add the remembered tags to the selected repository location or project.
