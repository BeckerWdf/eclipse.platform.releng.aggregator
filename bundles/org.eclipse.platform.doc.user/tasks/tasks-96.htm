Stream Resources

To add resources in a repository to the workbench:

1.

If the Repositories view is not already visible, then from the workbench window, select the Perspective > Show View > Other menu item.  Expand the Team category, select Repositories, then click the OK button on the dialog.

2.

There are 2 ways you can browse repository contents.  In the Repositories view, expand the repository location.  

If you are looking for a resource in a particular project version:

  • Expand the Project Versions category.

  • Find the project name that holds (or is) the resource you want to add to the workbench.  

  • Expand the project to reveal its versions.  You can expand each project version to view the resources it holds.

  • Select the resources you want to add to the workbench.

If you are looking for the latest resources in the stream:

  • Expand the Streams category.

  • Expand the stream which holds the resources you want to add to the workbench.

  • Select a resource in the stream to add to the workbench.

Note: You can Ctrl-click to select more than one stream resource or Shift-click to select a range.

3.

Right-click on the selected resources to bring up the context menu.  Select the "Add to Workspace" menu item.
