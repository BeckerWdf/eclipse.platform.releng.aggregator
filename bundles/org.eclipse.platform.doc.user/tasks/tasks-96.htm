Adding resources from a repository to the workbench

To add resources from a team repository to the workbench:

  1. Switch to the Team perspective or add the Repositories view to the current perspective.

  2. In the Repositories view, expand the repository location.

  3. Select the resources that you want to add to the workbench. If you are looking for a resource in a particular project version:
    1. Expand the Project Versions category and find the project name that holds the resource that you want to add to the workbench.  
    2. Expand the project to reveal its versions. You can expand each project version to view the resources it holds.

    If you are looking for the latest resources in the stream:

    1. Expand the Streams category.
    2. Expand the stream that holds the resources that you want to add to the workbench.
  4. From the pop-up menu for the selected resources, select Add to Workspace.

Tip: You can select Workbench > Preferences > Team from the main toolbar and select the radio button for Prompt for a location when adding projects from the repository.
