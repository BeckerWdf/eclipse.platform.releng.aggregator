Reverting to previously installed components

To revert a component to a previous component installation:

  1. Click Help > Software Updates. The Software Updates page opens.
  2. Click Revert. The Revert to a Previous Installation page opens.
  3. Select one of the installations in the history list. You can view the installed components in one of the installation histories by expanding it. Click Next.
  4. When the installation completion page opens, click Finish.
  5. Click OK to close the Software Updates page.
  6. Exit and restart the workbench to complete the install reversion.
