Revert to a Previous Installation

To revert a component to a previous component installation:

1.

From the workbench, select the Help > Software Updates menu item.

2.

In the dialog that comes up, click the "Revert..." button.  The revert to a previous installation page will appear.  

3.

Select one of the installations in the history list.  You can view the installed components in one of the installation histories by expanding it.
4.  Click the "Next" button of the wizard.
5.  Click the "Finish" button when the installation completion page appears.
6. Click the "OK" button of the Software Updates dialog. 
7. Exit the workbench and re-start the workbench in order to complete the install revision.
