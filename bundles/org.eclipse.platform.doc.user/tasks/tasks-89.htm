Running Ant scripts

To run an Ant script in the workbench:

  1. In the Navigator view, select an XML file.

  2. From the file's pop-up menu, select Run Ant. If the selected XML file is a valid script, the Ant wizard opens.

  3. Select a target from the Available Targets list. You can select more than one item in the list. The order in which you select the items is the order in which they will run. This is the number indicated in square brackets.

  4. If applicable, type any required arguments in the Arguments field.

  5. If you want output from the script to display in a console in the workbench, select Show execution log in console check box.

  6. Click Finish to close the wizard. If you selected the check box in the previous step, the Log Console displays any applicable execution results after the script runs.

Once a script is run through the Run Ant pop-up menu it is automatically configured as an external tool. In order to change the configuration or re-run the script, go to the Run > External Tools menu.
