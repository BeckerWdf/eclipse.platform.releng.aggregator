Comparing resources

When a comparison is performed, all applicable comparison editors appear in the editor area. In the comparison editors, the differences are highlighted so you can browse and copy changes between the compared resources.

To compare resources:

  1. Select two or three resources in the Navigator view.

  2. From the resource's pop-up menu, select Compare With.

The files are opened in the editor area, with differences highlighted. 

Here are some of the comparisons that you can perform. 

  • Compare With > Each Other - compares two or three source files in the Navigator view with each other.
  • Compare With > Local History - compares the selected resource to one that is in the local history, which is maintained when you save changes. (Tip: to configure the size and depth of your local history, select Workbench > Preferences.)
  • Compare With > Version - compares the selected resource with a version that has been released to the repository. This requires you to choose a version of the resource from a list.
  • Compare With > Stream Contents - compares the selected resource  with the version of the same resource that is currently released to the stream. 
  • Compare With > Base Version - compares the selected resource  with its state prior to modification, that is the version that you initially added from the repository.

Tip: You can customize the behavior of the comparison editor by selecting Workbench > Preferences > Workbench > Compare Viewers from the main menu bar. You can select to "lock scroll" the pane contents, as well as setting options for showing conflicts with other team members and changing the default font.
