Moving Using Drag and Drop

You can drag and drop to move resources from one workbench location to another (for example, from a project to another project).

You must drag the resource to the exact location in the Navigator view hierarchy where you want the file to reside.

1.

In the Navigator view, locate the resource that you want to move.

2.

Drag the resource to the desired location in the Navigator view. A horizontal location bar appears while you are still holding down the mouse button to indicate where the file will be placed.

3.

Notice that the resource is moved to the new location.
