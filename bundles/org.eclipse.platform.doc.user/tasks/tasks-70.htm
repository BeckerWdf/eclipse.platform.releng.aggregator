Compare with Local History

To compare a workbench resource with a state in the local history:

1.

In the Navigator view, select the resource for which you want to compare with a local history state.

2.

Bring up the context menu.  

3.

Select the Compare With > Local History... menu item.  
4. In the Local History list in the Compare with Local History dialog, select a state.

5.

In the Text Compare editor, click the Select Next Change and Select Previous Change buttons to browse the changes that have been made between the state in the local history and the workbench resource.

6.

Click the "Close" button to close the dialog.
