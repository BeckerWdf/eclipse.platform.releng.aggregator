Removing Components

To remove an installed component from the workbench:

1.

From the workbench, select the Help > Software Updates menu item.

2.

In the dialog that comes up, select the component you want to remove from the workbench.   

3.

Click the "Remove" button. 
4.  Click the "OK" button of the Software Updates dialog. 
5.  Exit the workbench and re-start the workbench in order to complete the uninstall.
