Adding line items in the Tasks view

The Tasks view contains line items for system-generated problems, warnings, and errors. You can add your own entries to the table, for example to build a list of to-do items, or tasks.

  1. On the toolbar in the Tasks view, click the New Task button . A new row is added to the table.

  2. In the row for the task that you just created, click in the Description field.

  3. Type a brief description for the task and press Enter.
