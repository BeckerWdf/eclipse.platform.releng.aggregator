Opening perspectives

Perspectives provide combinations of views and editors that are suited to performing a particular set of tasks. For example, you would normally open the Debug perspective to debug a Java program.

To open a new perspective:

  1. Click the Open Perspective button on the shortcut bar on the left side of the workbench window. (This provides the same function as the Perspective > Open menu on the menu bar.)

  2. (Optional) To see a complete list of perspectives, select Other from the drop-down menu.

  3. Select the perspective that you want to open. 

The title bar of the window changes to display the name of the perspective. An icon is added to the shortcut bar, allowing you to quickly switch back to that perspective from other perspectives.

Tips: 

  • To change the default behavior for where a new perspective opens, click Workbench > Preferences to change the default behavior.

  • To temporarily change the behavior, hold down one of the hotkeys defined on the workbench preferences page as you perform the following steps.
