Location for Added Projects

To enable prompting for a location for projects that are added to the workbench from a repository:  

1.

From the workbench, select the Window > Preferences menu item.

2.

In the preferences dialog, select the Team page in the left pane. 

3.

Select the checkbox labeled "Prompt for a location when adding projects from the repository".

Note: If this option is not enabled, projects will be automatically be placed in the default directory location.

4

Click the "OK" button of the preferences dialog.
