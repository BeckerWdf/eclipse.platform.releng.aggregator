Renaming Resources

You can rename workbench resources using the Rename command on the context menu in the Navigator view.

1. In the Navigator view, select the resource to rename.

2.

From the resource's context menu, select Rename.

3.

The Navigator overlays the resource's name with an text field.  Type in the new name and  press Enter.

 

If you decide you do not want to rename a resource, you can press Escape to dismiss the entry field.
