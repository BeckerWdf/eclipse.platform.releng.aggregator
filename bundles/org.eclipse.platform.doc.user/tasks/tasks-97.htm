Project Versions

If you want to be prompted for a location for projects that are added from the repository, you can enable a workbench preference (see Location for Added Projects).  If this option is not enabled, projects will be automatically placed in the default location.

1.

In the Repositories view, select a project version you want to add to the workspace.

2.

Bring up the context menu.  Select the Add to Workspace... menu item.

3.

During the add operation, a directory prompter will come up asking you to specify a location for the project.  Enter or select a directory.
