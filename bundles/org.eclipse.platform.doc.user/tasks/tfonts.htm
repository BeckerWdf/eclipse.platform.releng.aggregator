Setting help fonts and colors for accessibility

The help browser uses system settings for the font colors, styles, and sizes in the content. Users with visual impairments may wish to change some of these settings to increase the readability of the documentation.

To change font size and color settings:

  1. Open Microsoft Internet Explorer. The online documentation is displayed in a component of Internet Explorer, so changes you make to its display settings also affect the help display.

  2. Select Tools > Internet Options.

  3. On the General page, click the Accessibility button.

  4. Set the formatting options you desire.

  5. Optionally, you can specify a cascading style sheet (CSS) to apply to the content.

  6. Click OK and exit Internet Explorer.

  7. Restart the workbench. Open the Help perspective and browse the documentation to see the changes.

For more information about creating a CSS, consult a CSS reference. The W3 Consortium (www.w3.org) has an extensive collection of information about CSS and links to valuable resources.
