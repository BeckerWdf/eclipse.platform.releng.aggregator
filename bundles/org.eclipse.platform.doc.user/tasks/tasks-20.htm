Changing fonts

There are 3 main fonts in use by the Eclipse platform.

Banner Font
used in PDE editors, welcome pages and in the title area of many wizards. For instance the New Project wizard uses this font for the top title.
Header Font
used for the secondary titles in wizards. For instance the New Project wizard uses this font for the lower title.
Text Font
used in text editors and the console views.
You can change these fonts as follows:
  1. On the main menu bar, click Window > Preferences.
  2. Expand the Workbench category and select Fonts.
  3. Select the font you want to change.
  4. Click the Change push button to change the font.

Note: You can also click the Use System Font button to set the font to a reasonable value chosen by the platform. For example, on Windows this will use the font selected in the Display Properties control panel.
