Deleting resources

To delete a resource from the workbench:

  1. In the Navigator view, select the resources that you want to rename. (Hold down the Ctrl key to select more than one resource.)

  2. Press the Delete key.

You could achieve the same result by selecting Delete from the pop-up menu.
