Linking the Navigator view to the active editor

When you have multiple files open for editing, you can configure the Navigator view to automatically bring an open file to the foreground (make its editor session the active editor) every time you select that open file in the Navigator view. There are two ways to set this.

  1. From the Navigator menu, select Link With Editor.
  2. Click on the Link With Editor icon on the Navigator bar.
