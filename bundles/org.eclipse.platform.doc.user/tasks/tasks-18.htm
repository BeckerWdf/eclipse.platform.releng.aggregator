Linking Editor with Navigator View

You can choose whether to link the Navigator view's selection with the active editor (and vice versa) on the Workbench preferences page (Window > Preferences > Workbench).

1.

Select from the menubar Window > Preferences.

2.

In the left pane, select the Workbench page.

 

3.

Either select or clear the "Link Navigator selection to active editor" check box to indicate your preference.

  • If you select the check box, then selecting an open file in the Navigator view will activate the editor containing that file in the editor area and activating the editor changes the Navigation selection.
  • If you clear the check box, then selecting an open file in the Navigator view will have no effect on the editor area. In order to make an editor active, you will have to either click the editor's tab or double-click the file in the Navigator view.
