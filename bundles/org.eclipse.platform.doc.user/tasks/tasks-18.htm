Linking from the Navigator view to the active editor

When you have multiple files open for editing, you can configure the Navigator view to automatically bring an open file to the foreground (make its editor session the active editor) every time you select that open file in the Navigator view.

  1. From the main menu, select Window > Preferences.
  2. In the left pane, select the Workbench category.
  3. In the right pane, select Link Navigator selection to active editor.

If you leave the check box cleared, you will need to click tabs in the editor area or double-click open files in the Navigator view, in order to control which editor is active.
