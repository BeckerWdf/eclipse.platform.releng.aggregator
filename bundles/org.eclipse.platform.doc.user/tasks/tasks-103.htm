Replace With Repository Version

To replace workbench resources with versions in the repository:

 

1.

Select a resource in the Navigator view.

2.

Bring up the context menu.  Select one of the following menu items:

  • Replace With > Stream Contents: replace the workbench resource with the version currently released in the stream.
  • Replace With > Version: replace the workbench resource with a version in the repository to be selected by the user.
  • Replace With > Base Version: replace the workbench resource with the version that was taken from the stream during the last synchronization.

3.

In the case of "Replace With > Version", the Replace dialog comes up.  Browse the list to select the version with which you want to replace the workbench resource.  Notice you can see the differences between the version and the workbench resource in the text compare area.

Click the "Replace" button in the dialog to complete the operation.
