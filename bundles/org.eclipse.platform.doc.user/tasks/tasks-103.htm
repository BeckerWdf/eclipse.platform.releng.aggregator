Replacing resources with a repository version

To replace workbench resources with versions in the repository:

  1. Select a resource in the Navigator view.

  2. From the resource's pop-up menu, select one of the following menu items:

    • Replace With > Stream Contents - replace the workbench resource with the version currently released in the stream.
    • Replace With > Version - replace the workbench resource with a specific version that you select in the repository. When you select this option, another window opens, so that you can browse through versions in the repository and select the one that you want. The text compare area shows you the differences between the workbench resource and the various version in the repository.
    • Replace With > Base Version - replace the workbench resource with the version that was taken from the stream during the last synchronization.
  3. Click Replace to complete the operation.
