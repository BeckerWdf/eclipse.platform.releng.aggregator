Sorting Resources

To sort workbench resources in the Navigator view, select Sort from the view's pull-down menu. Select one of the following options:

  • By Name: The resources are sorted alphabetically by their full file name (e.g., A.TXT, then B.README, then C.HTML, etc.)
  • By Type: The resources are sorted alphabetically by file type/extension (e.g., all README files, then all TXT files, etc.)
