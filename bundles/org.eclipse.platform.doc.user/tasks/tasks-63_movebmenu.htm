Moving Using the Menus

You can move resources from one workbench location to another (for example, from one folder to another folder) using the Move command on the context menu in the Navigator view.

1.

In the Navigator view, select the resources that you want to move.

2.

From the context menu, select Move.

3.

In the Folder Selection dialog select the new location for the resources.  

4.

Select OK.
