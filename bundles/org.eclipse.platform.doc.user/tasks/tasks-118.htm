Versioning Stream Projects

You can version a project in the stream without adding the project to the workbench.  

1.

In the Repositories view expand a repository location and the Streams.  

2.

Select the stream that holds the project whose contents you want to version.  

Note: You can Ctrl-click to select more than one stream resource.

3.

Bring up the context menu.  Select the "Version From Stream..." menu item.

4.

In the Version Selected Resources dialog, choose one of the following version label options for the selected project (or projects):

  • Automatic: each project's version label is automatically generated 
  • One Name: the version label for each project is one name that is specified by the user
  • Name Each: for each project, the user is prompted to enter a version label

5.

Click the "OK" button on the versioning dialog when you have selected a version labeling scheme.
