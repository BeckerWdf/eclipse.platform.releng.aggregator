Versioning projects in the team stream

You can version a project in the stream without adding the project to the workbench.

  1. Switch to the Team perspective or add the Repositories view to the current perspective.

  2. In the Repositories view, expand a repository location and its streams.

  3. Select one or more streams that hold the projects whose contents you want to version.  

  4. Right-click and select Version from Stream from the pop-up menu. The Version Selected Resources window opens.

  5. Select one of the following version label options for the selected projects:

    • Automatic: each project's version label is automatically generated 
    • One Name: the version label for each project is one name that you specify
    • Name Each: for each project, you are prompted to enter a version label
  6. Click OK to close the window.
