Initial Version Label

You can change the initial version label that appears in the version selection dialog.  

1.

From the workbench, select the Window > Preferences menu item.

2.

In the preferences dialog, select the Team page in the left pane.

3.

In the "Initial version name" field, type the alphanumeric label that you would like to use as the initial automatically generated version label for projects that are versioned for the first time.
4.  Click the "OK" button of the Preferences dialog.
