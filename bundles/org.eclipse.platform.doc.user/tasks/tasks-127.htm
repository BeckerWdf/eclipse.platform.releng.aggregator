Adding a new update site

There are several different ways to add an update site to the list of sites that are used when browsing available software and checking for updates. Assuming you know the web address (URL) of the site that you want to add, you may add it in the following ways:

  • You may add the site using the Add Site... dialog.
    1. list of steps
  • You may drag and drop the site URL from a browser into one of the following drop targets:
    1. The Available Software page
    2. The Manage Sites... dialog
    3. The Location... text field in the Add Site... dialog.
