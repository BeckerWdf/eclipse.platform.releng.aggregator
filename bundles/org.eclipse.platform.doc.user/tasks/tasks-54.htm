Dragging from File System

You can use your operating system's file system browser to copy folders and files from the file system into the workbench.

Note: You must drag the resource(s) to the exact location in the Navigator view hierarchy where you want the resources to reside; you cannot simply drag and drop them onto a blank area in the Navigator view.

1.

Open your operating system's file system explorer.

2.

Locate the resource to import and drag it to a specific location in the Navigator view in the workbench.  

When dragging resources into the Navigator you will either see a horizontal line (as shown below) or the project/folder that you are trying to drop the resource into will be selected.

Drag the to the desired location and release the mouse button.

3.

Notice that the resource is copied into the workbench.
