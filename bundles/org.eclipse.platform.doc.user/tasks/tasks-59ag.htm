Exporting to a Zip File

You can use the export wizard to export resources from the workbench to a ZIP file in the file system.

1. In the Navigator select the resource to export.

2.

From the menubar, select File > Export...

3.

In the export wizard, select Zip file, then click Next.

4.

Using the left and right panes you can check the set of resources to export, or you can go with the default which is to export the entire resource you had selected.  

A grayed checkbox indicates that some, but not necessarily all, of their contents will be exported.

You can use the Select Types... button to filter the types of resources you want to export.

5.

In the Zip file field, type or browse to select the zip file where you want the exported resources to reside.

6.

In the Options area, you can choose to:

  • Overwrite existing resources without warning
  • Create directory structure for files
  • Compress the contents of the file

7.

Click Finish when you are done.
