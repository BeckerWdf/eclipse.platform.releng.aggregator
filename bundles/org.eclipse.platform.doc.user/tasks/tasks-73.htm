Building resources

There are number of ways that resources in the workbench can be built. The scope of a build can be one or more selected projects or over the entire workspace.

Builds can be full, meaning that all resources within the scope of the build are considered; or incremental, meaning that only resources that have changed since the last build are considered.

Builds can be done automatically (each time resources are modified), or manually, using a menu item. See the Related tasks links for more details.
