Building Resources

There are number of ways that resources in the workbench can be built.  The scope of a build can be over the entire workspace or one or more selected projects.  

Builds are full or  incremental; the former meaning that all resources within the scope of the build are considered, the latter meaning that only resources that have changed since the last build are considered.  

Builds can be done automatically (each time resources are modified), or manually, using a menu item.
