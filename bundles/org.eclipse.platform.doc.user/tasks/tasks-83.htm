Adding new tasks and types

It is possible to add new Ant tasks and types through the Ant preferences page. These tasks and types will be available for scripts running in the workbench without having to use taskdef or typedef in the script declaration (for more on taskdef or typedef see the Ant documentation in http://jakarta.apache.org/ant ). The following steps illustrate how to add a new task.

  1. Go to the menu Window > Preferences > External Tools > Ant.

  2. Select on the Tasks tab.

  3. Click on the Add Task button.

  4. Fill the task Name and Class.

  5. Select the Library where the task class is declared. If the library is not present on the list it has to be added to the Ant classpath.

Adding a type is very similar to adding a task and only requires small changes to the above steps.
