Changing Build Order

You can define the order in which the workbench performs builds on projects.

1.

From the workbench, select the Window > Preferences menu item.

2.

In the preferences dialog, select the Build Order page in the left pane.
3. Check the "Use Defaults" check box if you want to use the default build order, shown in the list.  Deselect this check box if you want to change the build order.

Note: you need to deselect this check box to see the build order list.

4.

If the "Use Defaults" check box is enabled, the buttons on this page are disabled.  To change the build order, you must first deselect this check box.

To change the build order in the list, select one or more projects in the list.  Use the buttons (to the left of the project list) to set the preferred project build order.

  • Up: moves the selection up in the list, making the selection closer to the beginning of the build order.
  • Down: moves the selection down in the list, making the selection closer to the end of the build order.
  • Add: allows you to add projects that are in the workbench that are not already in the build order.
  • Remove: removes the selection from the build order.

 
5.  Click the "OK" button of the Preferences dialog.
