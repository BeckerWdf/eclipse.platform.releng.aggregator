Updating components

There are two ways to update components installed in the workbench.  You can choose to update one or more selected components, or you can update all the installed components.  To update components installed in the workbench:

  1. Click Help > Software Updates. The Software Updates page opens.
  2. Select one of the Update buttons to open the New Component Updates page.

  3. To update one or more selected components, click Update.

  4. To update all components, click Update All.

  5. In the Location field, enter a valid location from which workbench components can be obtained.  (For example: http://siteaddr.domain/updates.) Optionally, add a description for the location in the Description field.
  6. Click Add to add the location to the list and then click Next. A page displaying the components opens.
  7. Select one or more components to install and click Next.
  8. Click Finish on the wizard, and click Finish again when the installation completion page appears.
  9. Click OK to close the Software Updates page.
  10. Exit and restart the workbench to complete the update.
