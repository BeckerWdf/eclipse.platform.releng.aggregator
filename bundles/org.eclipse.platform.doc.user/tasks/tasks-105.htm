Changing team settings

You can customize the settings related to a number of team views and operations.

  1. From the main menu bar, select Workbench > Preferences. 

  2. On the left side of the Preferences window, select the Team category.

Here are some examples of settings that you can change:

  • To change the initial version label for projects that are versioned for the first time, type an alphanumeric label in the Initial version name field.

  • To change the prefix label that appears in the Navigator view for outgoing changes, type an alphanumeric label in the Label prefix for outgoing changes field. (The prefix is shown only if you have Show Version Info selected in the Navigator view.)

  • To switch to the Team perspective when a synchronization operation is performed, select the Switch to team perspective on catch up/release checkbox.

  • If you want to be prompted for the location of projects added to the workspace from a repository, select the Prompt for a location when adding projects from the repository checkbox.
