Perform Automatic Builds

You can indicate whether or not the workbench should build resources automatically.

1.

From the workbench, select the Window > Preferences menu item.

2.

In the preferences dialog, select the Workbench page in the left pane.

3.

To enable automatic building, select the "Perform build automatically on resource modification" check box.  

  • If you choose this option, an incremental build of resources in the workspace will be performed each time a resource is modified.

To disable automatic building, de-select the "Perform build automatically on resource modification" check box.

  • If you choose this option, then you will have to perform builds manually, using the menu items provided.
4.  Click the "OK" button of the Preferences dialog.
