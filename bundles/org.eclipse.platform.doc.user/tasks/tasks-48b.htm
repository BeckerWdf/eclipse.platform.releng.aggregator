Showing or hiding files in the Navigator view

You can choose to hide system files or generated class files in the Navigator view. (System files are those have only a file extension but no file name, for example .classpath.)

  1. On the toolbar for the Navigator view, click the Menu button to open the drop-down menu of display options.

  2. Select Filter. 

  3. In the dialog box that opens, select the checkboxes for the types of files that you want to hide.
