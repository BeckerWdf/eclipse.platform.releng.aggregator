Outgoing Changes Prefix

You can change the prefix label for outgoing changes that appears in the Navigator view, when the Show Version Info menu item is enabled. 

1.

From the workbench, select the Window > Preferences menu item.

2.

In the preferences dialog, select the Team page in the left pane.

3.

In the "Label prefix for outgoing changes" field, type the alphanumeric label that you would like to use as the label that appears to the left of the file name in the Navigator.

Note: From the dropdown menu of the Navigator view, make sure the Show Version Info menu item is enabled (checked).   Other wise the outgoing changes prefix will not appear.
4.  Click the "OK" button of the Preferences dialog.
