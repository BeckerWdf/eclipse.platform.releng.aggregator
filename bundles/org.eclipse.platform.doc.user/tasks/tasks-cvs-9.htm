Checking out a module from a CVS repository

Modules can be defined in the CVS repository. To check out a module from a CVS repository to the workbench:

  1. Switch to the CVS Repository Exploring perspective or add the CVS Repositories view to the current perspective.

  2. Switch the CVS Respositories View to modules mode by selecting Show Modules in the drop down menu found in the title bar of the view.

  3. In the CVS Repositories view, expand the repository location.

  4. Expand HEAD and select the modules that you want to add to the workbench.
  5. From the pop-up menu select Checkout as Module. This will checkout the module into one or more projects in the local workspace, depending on how the module is defined in the repository.
