Importing from a Directory

You can use the import wizard to copy files from a file system directory into the workbench.

1.

From the workbench menubar, select File > Import...
2. In the import wizard, select File System, then click Next.

3.

The import from file system page is shown:

4.

In the Directory field click Browse... to select the directories from which you would like to add the resources.

5.

In the import selection panes, use the following methods to select exactly the resources you want to add:

  • Expand the hierarchies in the left pane and check or clear the boxes representing the containers in the selected directory. Then in the right pane, select or clear boxes for individual files.
  • Click the Select Types... button to filter the current selection for files of a specific type(s).
  • Click the Select All button to select all resources in the directory, then go through and deselect the ones that you don't want to add.
  • Click the Deselect All button to deselect all resources in the directory, then go through and choose individual resources to add.

6.

In the Where Do You Want The Imported Resources To Go? field, type or browse to select a workbench project or folder for the added resources.

7.

In the Options area, you can choose whether or not to:

  • overwrite existing resources without warning, and/or
  • create a complete file structure for the imported files.

8.

Click Finish when you are done.
