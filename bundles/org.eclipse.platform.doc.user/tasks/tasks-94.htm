Discarding a Repository Location

To discard a repository location:

 

1.

If the Repositories view is not already displayed, then from the workbench window select the Perspective > Show View > Other menu item.  Expand the Team category, select Repositories, then click on the OK button of the dialog.

2.

In the Repositories view, select the repository location you want to discard.

3.

Bring up the context menu.  Select the Discard Location menu item.

Note: You cannot discard a location if you have projects in the workbench that are shared with that location.
