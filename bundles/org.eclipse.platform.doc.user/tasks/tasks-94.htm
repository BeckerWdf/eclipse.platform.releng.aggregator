Discarding a repository location

Note: You cannot discard a location if you have projects in the workbench that are shared with that location.

To discard a repository location:

  1. Switch to the Team perspective or add the Repository view to the current perspective.

  2. In the Repositories view, select the repository location that you want to delete from the workbench.

  3. Select Discard Location from the pop-up menu.
