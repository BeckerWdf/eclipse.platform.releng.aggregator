Replace with Local History

To replace a workbench resource with a state in the local history:

1.

In the Navigator view, select the resource that you want to replace with a local history state.  

2.

Bring up the context menu.  

3.

Select the Replace With > Local History... menu item.  
4.  In the Local History list in the Replace from Local History dialog, select a state.

5.

In the Text Compare editor, click the Select Next Change and Select Previous Change buttons to browse the changes that have been made between states in the local history and the workbench resource.

6.

When you have found the state you want, click the "Replace" button.
