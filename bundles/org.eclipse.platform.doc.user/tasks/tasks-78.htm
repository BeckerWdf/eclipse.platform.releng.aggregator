Deleting Tasks

From the Marker Bar

You can delete associated tasks from the marker bar in the editor area.

1.

In the marker bar in the editor area, locate the task marker that you want to delete.  

2.

From the marker's context menu, select Remove Tasks.

3.

The task marker disappears, and the task is removed from the Tasks view.

From the Tasks View

You can delete one or more associated and unassociated tasks from the Tasks view.

1.

In the Tasks view select the tasks that you want to delete.

2.

Do one of the following:

  • Press the Delete key on the keyboard;
  • From the tasks' context menu, select Delete; or
  • Click the Delete button in the Tasks view's toolbar.

3.

Notice that the task markers no longer appear in the marker bar and that the tasks are removed from the Tasks view.

Completed Tasks

You can delete all completed tasks from the Tasks view.

1.

In the Tasks view list, access the context menu from anywhere in the view.

2.

Select Delete Completed Tasks.

3.

Notice that the completed tasks' markers no longer appear in the marker bar and that the tasks are removed from the Tasks view.
