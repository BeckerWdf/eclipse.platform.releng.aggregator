Specifying the Number of Installation Histories

To change the number of installation histories the workbench remembers:

1.

From the workbench, select the Window > Preferences menu item.

2.

In the preferences dialog, select the Software Updates page in the left pane. 

3.

Change the number in the text area labeled "Number of installation histories to keep".

4

Click the "OK" button of the dialog.
