Creating a new stream

Prerequisite: Before you create a new stream, you must have a valid repository location in the Repositories view.

To create a new stream:

  1. Switch to the Team perspective.

  2. From the pop-up menu for the Repositories view, select New > Stream. The New Stream wizard opens.

  3. From the Repository list, select the repository where you want the stream to reside.

  4. In the Name field, type the name for the new stream.

  5. Click Finish to create the new stream and close the wizard.
