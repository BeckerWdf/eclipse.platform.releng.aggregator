Creating a New Stream

To create a new stream:

 

1.

Pre-requisite: You must have a valid repository location in the Repositories View.

To bring up the New Stream wizard, do one of the following:

  • From the workbench, click the "Open The New Wizard" button on the toolbar; in left pane of the new wizard page select CVS; on the right pane, select Stream; click the "Next" button of the New wizard.
  • From the workbench, select the File > New > Other menu item; in left pane of the new wizard page select CVS; on the right pane, select Stream; click the "Next" button of the New wizard.
  • From the drop-down menu of the Repositories view, bring up the context menu; select the New > Stream menu item.

2.

From the Repository dropdown menu, select the repository where you want the stream to reside

3.

In the Name field, type the name for the new stream.

4.

Click the "Finish" button of the wizard.
