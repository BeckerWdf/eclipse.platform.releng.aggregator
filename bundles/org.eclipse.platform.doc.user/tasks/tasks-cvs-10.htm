Disconnecting a project from CVS

Disconnecting a project from CVS to disable the CVS operations that can be peformed on the project and it resources and optionally to remove the CVS information (stroed in the CVS folders) associated with the project.

To disconnect a project from CVS:

  1. In the Navigator view, select the project to be disconnected.

  2. Select Team > Disconnect... from the project's pop-up menu. The Confirm Disconnect from CVS dialog opens.

  3. In the dialog, choose one of:
    1. Delete the CVS meta information - disables the CVS team menu operations and removes the CVS folders and thier contents from the file system.
    2. Do not delete the CVS meta information - disables the CVS team menu operations but leaves the CVS meta information.

  4. Click OK to disconnect the project.
