Moving resources

You can move resources from one Workbench location to another (for example, from one project to another project).

  1. In the Navigator view, select the resources that you want to move.
  2. From the pop-up menu, select Move.
  3. In the Folder Selection window, select the project or folder where you want to move the resources and click OK.

Tip: You can also move resources by dragging them from the original location to the new location in the Navigator view.
