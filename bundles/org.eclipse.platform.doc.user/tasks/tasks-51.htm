Specifying File Editors

To associate editors with various file types in the workbench, follow these steps.

1.

Select from the menubar Window > Preferences.

2.

In the left pane, expand the hierarchy under Workbench, then select the File Editors page.

 

3.

Select an extension in the File types  list, or click Add... to add a new file type to the list.

Note: If you add a new file type, then you must type either a period followed by the file extension or an asterisk, period, and then the file extension (e.g., you can type either ".htm" or "*.htm", but not simply "htm").

4.

In the Associated editors list, select an editor to associate with that file type. If the editor you want is not in the list:

  • Click the Add ...button.
  • To associate an internal editor with this file type, select the Internal Editors button and choose the editor from the list.
  • To associate the file type with an editor external to the workbench, click the External Editors button and either select an application from the list or click Browse to select one from the file system registry.

Associating a File Type with an External Editor

If you associate a file type with an external editor in the workbench (Window > Preferences > Workbench > File Editors preferences page), then the workbench launches this external editor if you choose to open the file from the Navigator view.
