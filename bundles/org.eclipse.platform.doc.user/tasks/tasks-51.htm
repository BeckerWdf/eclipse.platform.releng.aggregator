Associating editors with file types

To associate editors with various file types in the workbench:

  1. From the main menu, select Workbench > Preferences.

  2. In the Preferences window, expand the Workbench category and select File editors.

  3. Select the file type from the Type list, or click Add to add a type that is not already on the list.

  4. In the Associated editors list, select the editor that you want to associate with that file type. To add an editor to the list:

    1. Click Add. The Editor Selection dialog box opens.
    2. Select Internal Editors or External Editors, depending on whether the editor that you want was built for the workbench or runs outside the workbench.
    3. If you select External Editors, you can click the Browse button to browse the system registry.
    4. Select the editor from the list and click OK.
  5. Click OK to finish associating the editor with the selected file type.

When you associate an internal editor with a file type, that editor opens in the editor area of the workbench, for example if you double-click a file in the Navigator or an entry in the Bookmarks or Tasks view.

Editors that support OLE document mode can also run in the workbench.

Tip: You can choose to override your default editor selections by selecting Open With from the pop-up menu for any resource in the Navigator view.
