Running external tools

The workbench provides a mechanism for running tools that are not part of the it. The following steps illustrate how to configure an external tool:

  • Go to the menu Run > External Tools > Configure.

  • Click on the New button.

  • Enter a name for your external tool (e.g. My External Tool).

  • Click on the Browse File System button.

  • Find the tool you want to run (e.g. in Windows it is usually a file with the extension .exe or .bat).

  • In the Tool Arguments field enter the necessary arguments for the tool (optional).

  • In the Working directory field enter the working directory for the tool (optional).

  • Click OK.

  • To run the tool go to the menu Run > External Tools > My External Tool.

It is also possible to use an external tool to build a project. Select the desired project, from its pop-up menu choose Properties > External Tools Builders and configure accordingly.
