Changing where perspectives open

You can change the default behavior for how perspectives are opened in the workbench. You can also redefine hotkeys for temporarily overriding the default behavior.

  1. Select Workbench > Preferences > Workbench.
  2. On the Workbench page, use the radio buttons to specify whether you want new perspectives to open in the same window or in a new window.
