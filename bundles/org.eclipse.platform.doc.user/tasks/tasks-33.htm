Managing Components

To view or change the components installed in the workbench:

1.

From the workbench, select the Help > Software Updates menu item.  In the dialog that comes up, a list of installed components is shown.  

2.

Use the buttons to the left of the list of components to manage the installed components:

  • Add: Allows you to add a new component to the workbench.  See Adding Components.

  • Remove: Un-installs one or more selected components from the workbench.  See Removing Components.

  • Update: Updates one or more selected components.  See Updating Components.

  • Update All: Updates all the components in the list to the most recent version.  See Updating Components.

  • Revert: Allows you to revert to a previous installation of a component.  See Revert to a Previous Installation.

3.

When you are finished, click the "OK" button of the dialog.
