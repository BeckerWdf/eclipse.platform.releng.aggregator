Managing components

To view or change the components installed in the workbench:

  1. Click Help > Software Updates. The Software Updates page opens, displaying a list of installed components.
  2. Click the buttons to the left of the list to add, remove, and update components, or to revert to a previous installation.
  3. Click OK when you are finished to close the Software Updates page.
