Exporting resources to a JAR file

You can use the Export wizard to export resources from the workbench to a JAR file in the file system.

  1. In the Navigator view, select the resources that you want to export.
  2. From the main menu bar, select File > Export. The Export wizard opens.
  3. Select JAR file and click Next.
  4. On the next page of the wizard, select the projects and packages that you want to export to a JAR file.
  5. Use the check boxes on the wizard page to specify various export options.
  6. Specify the path and name of the JAR file into which you want to export the selected resources.
  7. Click Finish.
