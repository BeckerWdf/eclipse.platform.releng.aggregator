Repositories view

The Repositories view, which is part of the Team perspective, shows the repository locations that you have added to your workbench. Expanding a location reveals the project versions and streams in that repository. You can further expand the project versions and streams to reveal the folders and files contained within them.

The pop-up menu for this view also allows you to specify new repository locations and add news streams to repositories. Use the Repositories view to add resources from the repository to the workbench, edit files in the repository, view resource history, and compare resource versions.

Like other views in the workbench, the Repositories view has its own toolbar. Toolbar buttons are provided for refreshing the view and for navigating.
