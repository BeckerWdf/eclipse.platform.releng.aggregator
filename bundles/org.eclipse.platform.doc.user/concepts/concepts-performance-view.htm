Performance view

The Performance view displays performance issues in Eclipse. Several Eclipse operations report on performance to the org.eclipse.core.runtime.PerformanceStats class when the time to perform an operation has met a threshold. Performance logging occurs if you run Eclipse with the -debug option. For further analysis, these performance issues are logged to an internal performance.log file in the .metadata directory of your Eclipse workspace.

To open the Performance view:

  1. Select Window > Show View > Other.
  2. In the Show View dialog select Basic > Performance.
