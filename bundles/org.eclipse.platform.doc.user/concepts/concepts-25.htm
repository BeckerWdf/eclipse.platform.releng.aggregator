Features

Independent Software Vendors (ISVs) add function to the workbench by delivering features, which are a mechanism for grouping plug-ins. Features, and thus all of their plug-ins, can be installed and uninstalled. Simple tools may consist of only one plug-in, while complex tools may require many.

The About item on the Help menu provides information about installed features and plug-ins. The Software Updates submenu on the Help menu groups together items for updating existing features, and for finding, downloading, and installing new features.
