Components

Independent Software Vendors (ISVs) add function to the workbench by delivering components, which are a mechanism for grouping plug-ins. Components, and thus all of their plug-ins, can be installed and uninstalled. Simple tools may consist of only one plug-in, while complex tools may require many.

The Update Manager item on the Help menu provides information about installed components.
