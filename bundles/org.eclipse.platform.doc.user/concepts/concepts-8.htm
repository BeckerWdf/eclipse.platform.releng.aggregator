Refreshing workbench data

You can change resources from within the workbench or outside the workbench. If you change a resource outside the workbench, you must refresh the workbench before you continue to work with resources in the workbench. This is also necessary if you modify resources using external editors. This will ensure that the resources in the workbench are up-to-date.

This is especially important when using the Team support.
