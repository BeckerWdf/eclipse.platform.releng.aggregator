Hierarchy Levels

Resources are stored and displayed in the workbench in hierarchies.  Described below are the terms used when referring to resources that are stored and displayed in a hierarchical structure.

Root

This term refers to the top level of the workbench contents (i.e., in the file system).

Parent Resource

This term refers to any resource that contains another resource. Only projects and folders can be parent resources. 

Child Resource

This term refers to any resource that is contained within another resource. Only files and folders can be child resources. This term is analogous to "child directories".
