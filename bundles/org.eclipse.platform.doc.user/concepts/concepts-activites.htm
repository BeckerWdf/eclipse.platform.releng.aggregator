Activities

Activities are plug-ins logically grouped to provide functionality and improve scalability. Within the "org.eclipse.activities" extension point, activities are declared in "plugin.xml" files.

Using the activities editor, it is easy to generate definitions as well as filter the activities that are required and hide those that are not needed. The activities editor edits XML files with the predefined title "activities.xml", in which all the user specified definitions are generated. Activities can be enabled or disabled using the preference page.

The activities editor filters plug-ins, but not contributions. It can also read "activity requirement bindings" on start up, however does not generate them.

The activities editor consists of three pages designed to assist in the creation and organization of these activities. It provides a tree-like representation of activities, and allows the visual addition and removal of categories, activities, plug-ins and their bindings.
