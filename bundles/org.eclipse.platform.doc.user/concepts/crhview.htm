Resource History view

This view provides a list of all the versions of a resource in the repository. From this view you can compare two versions, add a version to the workbench, or open an editor on a version.

Here is what the Resource History View looks like:

For each resource, the following is displayed in the table.

Version

This column displays the version name of the version in the history.  In parentheses, versions of the project that hold that version of the resource are listed.

Created

This column displays the creation date and time of the version in the history.

Author

This column displays the name of the person who created and authored the version.

Comment

This column displays the comment (if any) supplied for this version at the time it was released.
