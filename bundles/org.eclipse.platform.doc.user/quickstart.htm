Quick Start Guide

This tutorial introduces you to the Workbench, providing a brief overview of the steps required to create simple applications.

 

Task Description

Create a Project

 

 
  1. Select File > New > Project....
  2. In the New Project wizard, select Simple > Project then click Next.
  3. In the Project name field, type a name for your new project.

  4. Specify a location for the project or use the default.

  5. Click Finish.

Create a Folder

 

 
  1. Select File > New > Folder.
  2. In the Folder name field of the New Folder wizard, type a name for your new folder.
  3. Click Finish.

Create a File

 
  1. Select File > New > File.
  2. In the File name field of the New Folder wizard, type a name for your new folder.
  3. Click Finish.

Import Resources

 

 
  1. Select File > Import.
  2. Select an import source.
  3. Click Next.
  4. Select the directory containing the resource by following the steps indicated in the applicable wizard.
  5. Click Finish.

Export Resources

 

 
  1. Select File > Import.
  2. Select an import source.
  3. Click Next.
  4. Select the directory containing the resource by following the steps indicated in the applicable wizard.
  5. Click Finish.
