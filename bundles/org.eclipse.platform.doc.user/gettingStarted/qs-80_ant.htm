Ant & External Tools tutorial

This chapter covers Eclipse's External Tools framework, and its integration with Ant, the build tool from the Apache Software Foundation. If you are not familiar with Ant, please look here for documentation. This chapter assumes you have a basic knowledge of Ant.

This chapter is split into three main sections. First, we cover the basics of working with Ant scripts in Eclipse. Next, we work through several Eclipse use cases in which Ant scripts can make life a little easier. Finally, we consider the entire External Tools framework and how to use non-Ant tools.
