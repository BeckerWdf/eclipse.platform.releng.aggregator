Starting a search

You can search for text strings in the workbench as follows:

1.

In the workbench toolbar, click the Search button .

2.

The File Search page should be displayed. If this is not the case press the corresponding tab to bring the page to front. If the tab itself is not available you can add it to the Search dialog by clicking the Customize... button. The Search Page Selection dialog allows you to configure which pages are shown in the Search dialog.

3.

In the Containing text field, type in: it.

The combo box for the Containing text field also lets you select from a list of recently performed searches.

4.

You can select or deselect the Case sensitive check box depending on whether or not you want to perform a case sensitive or insensitive search. 

We want to search for lowercase "it" so check the Case sensitive box.

5.

Select the scope of your search in the Scope part of the dialog. You can search the entire workspace, pre-defined working sets or the previously selected resources.
For this example we search the whole workspace.

6.

In the File name patterns field you can specify the file name patterns of the files which will be include in your search. 

Click Browse...  to open the Select Types dialog. This dialog provides you with a quick way to select from a  list of registered extensions.

 For the moment we will confine our search to .txt files. Ensure .txt is checked and click OK.

7.

Click Search.

Note that you can click Cancel to cancel the search while it is in progress and that the Search dialog shows the progress of the search.
