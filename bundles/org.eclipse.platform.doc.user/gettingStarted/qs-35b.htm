Copying

Let's start by copying JanesFile.txt to our new  folder (JanesOtherFolder). Once we have copied our file we will then rename it.

1.

Ensure that you have performed the setup described in the introduction to this section. 
2. In the Navigator view, select JanesFile.txt

3.

From the file's context menu, select Copy (or Ctrl+C)

4.

In the Folder Selection dialog, select JanesOtherFolder as the destination.

Note: If you select the same folder as the original, the copy will be renamed so that it will not conflict with the original file.

5.

In the Navigator view expand JanesOtherFolder and confirm that you now have a second copy of JanesFile.txt.

 We have seen how to copy files using the copy operation. It is also possible to copy files by holding down the Ctrl key while dragging a file from one folder to another folder.
