Setting up a CVS repository

A repository is a persistent store that coordinates multi-user access to the resources being developed by a team. The Workbench comes with CVS support built-in. You can find more information on CVS at https://directory.fsf.org/wiki/CVS.

Refer to this site for information on how to install and configure a CVS repository including user access and passwords.

In order to continue with this tutorial, access to a CVS repository is required.
