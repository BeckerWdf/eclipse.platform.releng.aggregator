Team CVS tutorial

In this chapter, we will learn to use the CVS team capabilities built into the workbench. The steps in this chapter are team oriented; you cannot (easily) complete these tutorials if others are not simultaneously working through this chapter.

We will learn how to work on our project, then commit changes to the repository for others on the team to use. As we continue to work on the project alongside other users we will see how to Commit resources that other users may be simultaneously working on, and how to results those changes in the project.

Remember, before we get started you need to entice at least one coworker into working through these steps with you.
