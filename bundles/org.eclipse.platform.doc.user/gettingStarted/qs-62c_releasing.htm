Releasing

Now that we have a better understanding of the Synchronize view let's go ahead and release the work we have done.

1.

In the Structured Compare list select file1.txt

2.

Popup the menu and choose Release. Click OK when asked to supply a release comment.

3.

Observe that file1.txt is no longer in the list.

In addition to releasing individual resources we can also select the project and release all of the changes in the project.
4. Select JanesTeamProject.
5. Choose Release from the project's context menu and type in "my first project" when prompted for a release comment.

Observe that the remaining resources in the project are released and the structure compare list is now empty.
