Drag and drop or copy and paste

You can use your operating system's file system explorer to export a copy of a folder or file from the workbench to the file system.

1.

Open your operating system's file system explorer.

2.

Drag the file JanesFile.txt from the Navigator to the file system explorer.

3.

Notice that the file is copied to the destination in the file system.

4.

The export can also be achieved by selecting the file in the Navigator and choosing Edit > Copy, then

pasting it in the file system explorer.
