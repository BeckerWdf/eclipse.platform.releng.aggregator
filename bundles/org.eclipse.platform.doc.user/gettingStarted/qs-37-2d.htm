Unassociated tasks

Unassociated tasks are not associated with any specific resource. Create an unassociated task as follows.

1.

In the Tasks view, click the New Task button .

2.

A new task appears in the Tasks view.

3.

In the row for the new task that you've just created, click in the Description field.

4.

Type a brief description for the task and press Enter. If you change your mind while entering the description you can press Escape to cancel the edit area.
