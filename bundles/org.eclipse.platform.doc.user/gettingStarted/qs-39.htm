Rearranging views & editors

We have seen how to work with editors and views. In this section we will learn how to rearrange these views and editors to customize the layout of the workbench.

Setup

Before we start rearranging our workbench, we need to do a little house keeping.

1.

Start by choosing Perspective > Reset and selecting OK. This will reset the current perspective to its original views and layout.

2.

Ensure there are editors open for JanesFile.txt and JanesText.txt. Close any other open editors including the Welcome Page if it is still open. Don't worry. You can always get back to the welcome page by choosing Help > Welcome.
